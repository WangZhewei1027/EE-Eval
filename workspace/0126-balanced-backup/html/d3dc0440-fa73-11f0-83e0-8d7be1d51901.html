How it works (short)

A feedforward neural network maps input -> hidden layers -> output. Each connection has a weight; nodes add a bias and apply an activation function (sigmoid/tanh/ReLU). Training adjusts weights by computing gradients of a loss (difference between predictions and labels) using backpropagation and gradient descent. This demo trains a small network on 2D points and shows the decision boundary (color map), node activations (node color), and weight magnitudes (line widths).
