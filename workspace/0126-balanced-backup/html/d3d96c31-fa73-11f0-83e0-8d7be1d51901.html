Quick Reference & Examples

Common SQL statements

  • SELECT — read rows: SELECT col1, col2 FROM table WHERE condition ORDER BY col LIMIT n
  • INSERT — add rows: INSERT INTO table (col1,col2) VALUES (v1,v2)
  • UPDATE — modify rows: UPDATE table SET col = expr WHERE condition
  • DELETE — remove rows: DELETE FROM table WHERE condition
  • JOIN — combine tables: SELECT ... FROM a JOIN b ON a.id = b.a_id
  • GROUP BY — aggregate: SELECT col, COUNT(*) FROM table GROUP BY col HAVING COUNT(*) > 1
