Encrypt & Decrypt

Anyone can encrypt messages with your public key, but only you can decrypt them with your private key.

How Asymmetric Cryptography Works:

  • Public key encrypts, private key decrypts (confidentiality)
  • Private key signs, public key verifies (authentication)
  • No need to share secret keys before communication
  • Used in SSL/TLS, PGP, SSH, cryptocurrencies, etc.
