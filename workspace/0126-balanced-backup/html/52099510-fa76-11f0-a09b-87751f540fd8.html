PageRank Algorithm

PageRank is a link analysis algorithm developed by Larry Page and Sergey Brin at Stanford University in 1996.

How it works

PageRank is calculated using a combination of three factors: the number of outgoing links from each page, the link weight (or PageRank score), and the number of links on the page.

Here's a step-by-step explanation:

  1. Start with a single page (P0) with a PageRank score of 1.
  2. For each page (i, 1 ≤ i ≤ N), calculate the PageRank score of each link (j) pointing to page (i) as follows:
  3. PageRank(i, j) = (1/N) \* (1 - PageRank(j, i)) + (N - 1) \* PageRank(j, i)
  4. PageRank(i, i) = 1
  5. PageRank(i, j) = PageRank(j, i) \* LinkWeight(j, i)

Repeat this process until convergence.
