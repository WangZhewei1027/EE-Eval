Linked List (Singly) Visualizer

Interactive demo to learn how a singly linked list works. Add, insert, remove, find or reverse the list and watch the nodes and pointers update.

Tips: Index 0 refers to head. Removing an absent value will do nothing. Reverse mutates the list.
