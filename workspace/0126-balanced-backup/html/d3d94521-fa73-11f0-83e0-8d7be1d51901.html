- Virtual addresses are split: Page number + Offset. Page size determines offset bits.
- Page table maps each virtual page to a physical frame (or invalid). A page fault happens when a page is not in memory.
- Replacement policy decides which resident page to evict when frames are full.
- TLB caches recent translations for faster hits.
