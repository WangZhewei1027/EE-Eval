Backtracking: Sudoku Solver

How Backtracking Works:

This Sudoku solver uses backtracking algorithm:

  1. Find an empty cell in the Sudoku grid
  2. Try placing digits 1-9 in that cell
  3. For each digit, check if it's valid (no conflicts in row, column, or 3x3 box)
  4. If valid, place the digit and recursively try to fill the rest of the grid
  5. If no digit works, backtrack (undo the previous choice) and try a different digit
  6. Repeat until the grid is filled or all possibilities are exhausted
