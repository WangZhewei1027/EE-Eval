Recursion

Recursion is a programming technique where a function calls itself repeatedly until it reaches a base case or reaches an end condition. This can be useful in situations where we want to repeat a certain operation multiple times, such as iterating over an array or searching for a specific value.

The basic syntax of recursion is:

function callFunction(baseCase, arguments) { if (baseCase === arguments.length) { return arguments; } return callFunction(baseCase + 1, arguments); } var arr = [1, 2, 3]; var result = callFunction(0, arr); document.write(result); // Output: [1, 2, 3]

  • Given a list of numbers, find the sum of all the numbers in the list.
  • Given a string, find its length and return the number of characters in the string.
