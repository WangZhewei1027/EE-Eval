How it works (Max Heap):
  • Binary heap stored as array: for index i, parent = Math.floor((i-1)/2), children = 2*i+1 and 2*i+2.
  • Insert: place element at end and sift-up while greater than parent (swap).
  • Extract max: swap root with last, pop max, then sift-down from root to restore heap.
  • Heapify (bottom-up): run sift-down from floor(n/2)-1 down to 0 for O(n) build.
Interactive demo — values highlighted during compare/swap. Small delays show step-by-step operations.
