How it works

- Each task has simple instructions (INC, ADD, PRINT, YIELD, JUMP)
- CPU executes up to "Quantum" instructions then performs a context switch (preemptive), or only on YIELD (cooperative)
- Context switch saves task registers (PC, A, B) and simulates overhead cycles
- Try different overhead and quantum to see throughput and switching frequency
