BFS (pseudocode)
BFS(start):
  queue ← [start]
  visited[start] ← true
  while queue not empty:
    v ← queue.pop_front()
    for each neighbor u of v:
      if not visited[u] and not wall:
        visited[u] ← true
        parent[u] ← v
        queue.push_back(u)
        if u == target: reconstruct path and stop
Tip: Use "Step" to execute one BFS operation at a time and watch the queue update. Resize the grid, create mazes, and observe how BFS finds the shortest path in an unweighted grid.
