Heap Sort Explanation:

Heap Sort is a comparison-based sorting algorithm that uses a binary heap data structure. It has two main phases:

  1. Heapify: Build a max heap from the input data.
  2. Sorting: Repeatedly extract the maximum element from the heap and rebuild the heap.

Time Complexity: O(n log n) in all cases.

Space Complexity: O(1) as it's an in-place algorithm.
