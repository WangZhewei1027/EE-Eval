Example of Floyd-Warshall Algorithm

The Floyd-Warshall algorithm is used to find shortest paths between two points on a graph represented by nodes and edges. It works by maintaining a list of all possible pairs of nodes that can be connected to the input node, and then checking which pair has the smallest distance from the input node.
