Circular Linked List — Interactive Demo

A circular linked list is like a normal linked list, but the last node's next pointer points back to the head, forming a circle. Use the controls to create, modify and step through the list.

Head ➜ marked
Arrows show next pointers
Single node forms a self-loop
List is empty
Traversal controls: "Start / Next" steps through nodes in order. "Animate" will loop through all nodes automatically once.
Notes:
  • Indexing for operations is 0-based (head is index 0).
  • When the list is empty, insertHead and insertTail create the first node.
Quick reference (time complexity):
Insert head: O(1) if tail known else O(n)
Insert tail: O(1) if tail known else O(n)
Search / Delete: O(n)
