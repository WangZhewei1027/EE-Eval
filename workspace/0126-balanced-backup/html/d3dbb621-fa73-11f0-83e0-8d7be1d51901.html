Support Vector Machine — Interactive Demonstration (Pegasos SGD)

How to use
- Click on the canvas to add points. Use "Class" buttons to pick label.
- Press "Train" to run the SVM learner (Pegasos). "Epochs" controls training iterations.
- Toggle "Auto-train" to update continuously while adding points.
- "Random dataset" generates sample data; "Clear" removes all points.
- Support vectors are highlighted with black outlines when hinge loss is active (y*(w·x+b) ≤ 1).
