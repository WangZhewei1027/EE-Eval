Interactive Logistic Regression Demo

This interactive demo shows binary logistic regression on 2D points. Click to add points, generate datasets, and train a logistic model (weights updated by gradient descent). The background shows predicted probability (blue = class 1, red = class 0). The decision boundary is drawn where p = 0.5.

Weights: [0, 0]
Bias: 0

How it works

Logistic regression models the probability that a point (x,y) belongs to class 1:

p = σ(w1*x + w2*y + b), where σ(z) = 1 / (1 + e^-z)

We minimize the average cross-entropy loss:

L = -1/N Σ[y log p + (1-y) log(1-p)] + (λ/2) * ||w||^2

Gradient descent updates the weights w, b using the average gradient (p - y) times inputs. Use the controls to change the learning rate and regularization. Click the canvas to add points (choose class above).

Tips

  • Generate "separable" to see near-perfect classification.
  • Increase learning rate to speed training; too large may diverge.
  • Regularization helps when data overlaps (prevents large weights).

Implementation notes

Logistic regression implemented with full-batch gradient descent. Canvas used to draw probability heatmap (background), points, and decision boundary (p=0.5).
Left-click on the plot to add points. Choose which class to add using the radio buttons. Right-click is disabled for adding.
