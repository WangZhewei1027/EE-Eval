About Tim Sort

Tim Sort is a hybrid sorting algorithm derived from Merge Sort and Insertion Sort. It was designed to perform well on many kinds of real-world data.

Key characteristics:

  • Works by dividing the array into small runs (subarrays)
  • Uses Insertion Sort for small runs (typically 32-64 elements)
  • Merges runs using a method similar to Merge Sort
  • Stable sort (maintains relative order of equal elements)
  • Best for: Partially sorted arrays, real-world data
  • Time complexity: O(n log n) in worst case, O(n) in best case
  • Space complexity: O(n)

This is the algorithm used in Python's built-in sort() method and Java's Arrays.sort().
