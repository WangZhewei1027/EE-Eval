Algorithm & Explanation

Pseudocode (linear search) 
function linearSearch(array, target):
  for i from 0 to array.length - 1:
    if array[i] == target:
      return i   // index where target found
  return -1      // not found
Time Complexity
Best case: O(1) — target at first element.
Average case: O(n) — target somewhere in the array or not present.
Worst case: O(n) — target at end or not present.
Space complexity: O(1) — constant extra space.
Notes
  • Linear search works on unsorted arrays.
  • For sorted arrays, binary search is faster (O(log n)).
  • This demo supports finding the first match or all matches.
Example uses
  • Small arrays where overhead of sorting/searching algorithms isn't worth it.
  • When data isn't ordered or there's no quick index lookup.
