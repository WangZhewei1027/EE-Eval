Dynamic Typing in JavaScript

What is Dynamic Typing?

Dynamic typing means that variables can hold values of any type, and their type can change during runtime. Unlike statically-typed languages where you declare a variable's type, JavaScript variables can be reassigned to different types.

Example 1: Variable Type Change

let dynamicVar = 42;          // Number
dynamicVar = "Hello";       // Now a String
dynamicVar = true;          // Now a Boolean

Example 2: Type Coercion

JavaScript will automatically convert types in certain operations (called type coercion):

let result = "5" + 2;       // "52" (string concatenation)
let anotherResult = "5" - 2;  // 3 (numeric subtraction)

Example 3: Checking Types

Use typeof to check a variable's current type:

let item = 100;
console.log(typeof item);  // "number"

item = "Hello";
console.log(typeof item);  // "string"

Example 4: Loose Equality (==) vs Strict Equality (===)

Loose equality performs type coercion, strict equality doesn't:

"5" == 5    // true (type coercion)
"5" === 5   // false (different types)
