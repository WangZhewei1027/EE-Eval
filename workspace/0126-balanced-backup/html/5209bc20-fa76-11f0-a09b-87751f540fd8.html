Greedy Algorithm Demo

Greedy Algorithms are a type of algorithmic problem-solving strategy that involves making the locally optimal choice at each step in a sequence in order to make a globally optimal solution. This is often contrasted with the "happier" algorithm, which makes a solution that is good for all possible next moves.

Example: Coin Change Problem

Given a set of coins and a target amount, find the fewest number of coins needed to reach the target amount.
