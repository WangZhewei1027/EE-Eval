Queue Example

Here is an example of how to create a queue using JavaScript:

```javascript var queue = []; queue.push('item'); queue.push('item'); queue.push('item'); queue.push('item'); console.log(queue); // Output: ['item', 'item', 'item'] ``` In this example, we first define a variable called `queue` and add three elements to it. Then, we use the `push()` method to push each element into the queue. When we run this code, the output will be: ``` ['item', 'item', 'item'] ``` This shows that the queue has now been created with the elements 'item', 'item', and 'item'. When we call the `pop()` method on the queue, it will return the last element in the queue, which is also 'item' in this case.
