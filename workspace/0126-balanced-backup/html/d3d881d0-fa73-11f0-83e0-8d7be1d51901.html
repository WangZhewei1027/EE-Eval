Time Complexity — Interactive Demo

Explore how different algorithms scale as input size grows. The demo counts "elementary operations" (a rough proxy for work) and optionally measures elapsed time. Use controls to change algorithm and range, then plot or run single tests.

Operation counters are an illustrative approximation of "work" and are counted inside each algorithm's key steps (comparisons, assignments, loop iterations). Timing results depend on your browser and machine.
