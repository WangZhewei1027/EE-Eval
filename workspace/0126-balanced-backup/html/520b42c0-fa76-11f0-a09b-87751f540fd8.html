Refactoring Example

This is an example of refactoring. We will refactor the code to make it cleaner and easier to maintain.

Before Refactoring:

            
                var data = {
                    "users": [
                        {"id": 1, "name": "John Doe", "email": "john@example.com"},
                        {"id": 2, "name": "Jane Doe", "email": "jane@example.com"}
                    ]
                };

After Refactoring:

            
                var users = {
                    "id": 1,
                    "name": "John Doe",
                    "email": "john@example.com"
                };
