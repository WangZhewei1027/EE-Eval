DNS (Domain Name System) Demonstration

DNS is like the phonebook of the internet. It translates human-friendly domain names (like example.com) into IP addresses (like 93.184.216.34) that computers use to identify each other.

DNS Lookup Process

1. You enter a domain name in your browser
2. Browser checks its cache for the IP address
3. If not found, the OS checks its hosts file and cache
4. Query sent to the configured DNS resolver (usually your ISP)
5. Resolver checks its cache, if not found queries root servers
6. Root server directs to the appropriate TLD (Top-Level Domain) server
7. TLD server directs to the authoritative name server for the domain
8. Authoritative server provides the IP address
9. The IP address is returned to your computer
10. Your browser connects to the IP address
