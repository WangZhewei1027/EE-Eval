Multiset

A multiset is a set in which each element is associated with a set of indices. It is used to represent data that has multiple values for each unique element.

Example 1

Let's consider a multiset with the following elements: {a: 2, b: 3, c: 1}

Count: {a: 2, b: 3, c: 1}

Example 2

Let's consider a multiset with the following elements: {x: 2, y: 3, z: 1}

Count: {x: 2, y: 3, z: 1}
