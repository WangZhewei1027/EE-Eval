Quick notes
  • Root servers know only TLD servers. TLD servers know authoritative servers for second-level domains (example.com).
  • Recursive resolution: stub asks resolver, resolver does the work and returns final answer.
  • Iterative resolution: resolver may return referrals; this demo highlights the steps so you can watch what happens.
  • CNAME is an alias: an extra query resolves the canonical name.
