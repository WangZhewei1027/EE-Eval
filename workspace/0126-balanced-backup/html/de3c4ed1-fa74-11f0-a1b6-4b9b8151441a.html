Divide and Conquer: Merge Sort

Merge Sort is a classic example of the Divide and Conquer algorithm paradigm. It works by recursively dividing the input array into smaller subarrays until each subarray contains a single element, then merging those subarrays in a sorted manner.

How Merge Sort Works

Divide: The array is divided into two halves recursively until each subarray has only one element.
Conquer: Each single-element subarray is trivially sorted.
Combine: The sorted subarrays are merged back together in a way that maintains the sorted order.

The time complexity is O(n log n) in all cases, making it more efficient than simpler algorithms like Bubble Sort for large datasets.
