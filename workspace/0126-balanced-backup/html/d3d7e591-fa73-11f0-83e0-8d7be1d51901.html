Dynamic Programming (DP) is a set of techniques for optimizing problems that have overlapping subproblems and optimal substructure. Common patterns: top-down memoization (recursion + cache) and bottom-up tabulation (iteratively filling a table).

Built for learning — explore how DP reduces repeated work and how tables represent subproblem solutions.
