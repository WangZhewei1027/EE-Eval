Dynamic Array — Interactive Demonstration

See how an array resizes automatically as you append, insert, and remove elements. Watch capacity doubling and copying.

A dynamic array provides array-like indexing with automatic resizing. Internally it keeps a capacity and a size. When you append beyond capacity the array allocates a larger block (commonly double) and copies existing elements into it. This demo simulates that process so you can observe the cost and the amortized performance.

Pseudocode (simple)
function append(A, value): if A.size == A.capacity: newCap = max(1, A.capacity * 2) B = allocate(newCap) for i in 0 .. A.size-1: B[i] = A[i] # copy element A.data = B A.capacity = newCap A.data[A.size] = value A.size += 1
70ms
Tip: Append repeatedly to trigger resizing. Watch how many elements are copied during each resize — resizing is O(n), but amortized cost per append is O(1).
