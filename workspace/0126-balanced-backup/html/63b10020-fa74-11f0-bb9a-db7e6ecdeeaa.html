Selection Sort Visualization

Selection Sort: This simple comparison sorting algorithm works by repeatedly finding the minimum element from the unsorted part of the array and moving it to the beginning. Here's the process:
  1. Start at the first element (consider it minimum).
  2. Compare it with the rest of the array to find the true minimum.
  3. Swap the found minimum element with the element at the current position.
  4. Move to the next element and repeat until the entire array is sorted.
