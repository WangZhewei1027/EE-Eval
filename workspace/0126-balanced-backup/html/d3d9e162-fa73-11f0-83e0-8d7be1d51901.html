Concepts demonstrated:
  • Request line: METHOD path HTTP/version
  • Headers: key: value pairs, case-insensitive
  • Status line: HTTP/version status reason
  • Transfer-Encoding: chunked — streaming response
  • Set-Cookie and subsequent Cookie header
  • ETag and conditional GET → 304 Not Modified
Tip: Try POST /echo with JSON body, enable chunked streaming for /stream, or visit /redirect to see 301/302 flows.
