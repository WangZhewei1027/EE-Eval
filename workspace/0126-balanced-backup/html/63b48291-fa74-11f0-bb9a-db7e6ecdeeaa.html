This is a simple demonstration of a feedforward neural network with 2 input neurons, 2 hidden neurons, and 1 output neuron. Use the sliders above to change input values. The network computes its output and displays it below.

Clicking Train Network runs a few iterations of training using the AND logic gate as the target function. Watch how the network adjusts its weights to correctly model the AND gate.

The weights are visualized as connections between neurons; thicker and darker lines mean stronger weights. Hover over neurons to see their current activation values.
