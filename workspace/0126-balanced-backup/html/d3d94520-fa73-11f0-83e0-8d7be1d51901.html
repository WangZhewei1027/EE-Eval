Help & explanation
This simulator models a simple paging virtual memory system:
  • Virtual addresses = page number + offset (page size configurable).
  • A TLB caches recent page→frame translations (optional).
  • On access: check TLB → page table. If the page isn't resident, a page fault occurs and the page is loaded into a physical frame. If no free frame, an existing page is evicted using the selected replacement algorithm. Writes set the page dirty and must be swapped-out if evicted.
  • Use Step to advance one access, Run to replay the sequence automatically.
Built for learning: virtual pages, page tables, TLB, page faults and replacement.
