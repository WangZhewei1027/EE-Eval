Big-Omega Notation Demonstration

Big-Omega notation (Ω) provides a lower bound for the runtime of an algorithm. It describes the best-case scenario for the growth rate of an algorithm's runtime. Below is an example demonstrating Big-Omega notation through a simple algorithm.

JavaScript Function: 
function findMin(array) {
    if (array.length === 0) return null; // Ω(1) - constant time for empty array
    let min = array[0];
    for (let i = 1; i < array.length; i++) {
        if (array[i] < min) {
            min = array[i];
        }
    }
    return min; // Ω(n) - linear time in the worst case
}
