Socket Programming Demo

Socket Programming Concepts — WebSocket, WebRTC DataChannel & Virtual Sockets

This single-page demo illustrates core socket programming concepts: - how clients and servers exchange data over persistent connections, - different socket types: TCP-like streams, WebSockets (browser ↔ server), and WebRTC DataChannels (peer-to-peer), - message framing, fragmentation, binary vs text data, and connection lifecycle events. The demos are interactive; try the WebSocket echo server (real network), and a local WebRTC DataChannel that simulates two peers inside your browser. There's also a simple virtual socket that shows fragmentation and reassembly logic you would implement in a real TCP/UDP app.

Virtual Socket — Simulate Fragmentation & Framing

This demonstrates how a length-prefixed framing protocol reassembles fragmented TCP messages. You can send a long message, configure a small MTU to force fragmentation, and see how the receiver reassembles complete logical messages.

Length-prefixed framing (concept)
Before sending: prefix each message with a fixed-size length header (e.g. 2 or 4 bytes). The receiver reads bytes until the length header is complete, then reads the declared payload size, even if it arrives in fragments. This demo uses a 2-byte big-endian length prefix.

Server & Client (Node.js) — Raw TCP Example

The browser cannot open raw TCP sockets, but below is a small Node.js example using the net module showing how a TCP server and client work. Run on your machine to test real sockets. 
/* TCP server (Node.js)
const net = require('net');
const server = net.createServer(socket => {
  console.log('client connected', socket.remoteAddress);
  socket.on('data', data => {
    console.log('received:', data.toString());
    // echo back
    socket.write(Buffer.from('Echo: ' + data.toString()));
  });
  socket.on('close', () => console.log('client closed'));
});
server.listen(9000, () => console.log('listening on 9000'));


/* TCP client (Node.js)
const net = require('net');
const client = net.createConnection({ port: 9000 }, () => {
  console.log('connected to server');
  client.write('Hello server!');
});
client.on('data', data => {
  console.log('server says:', data.toString());
  client.end();
});
client.on('end', () => console.log('disconnected'));
*/
Note: WebSocket demo uses public echo server wss://echo.websocket.events (may be rate-limited).
