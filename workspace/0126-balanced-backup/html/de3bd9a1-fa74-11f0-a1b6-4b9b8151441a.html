What is Interpolation Search?

Interpolation search is an improved variant of binary search for uniformly distributed, sorted arrays. Instead of always dividing the search space in half, it estimates the position of the target value based on the values at the boundaries.

The formula used is: pos = low + ((target - arr[low]) * (high - low)) / (arr[high] - arr[low])
