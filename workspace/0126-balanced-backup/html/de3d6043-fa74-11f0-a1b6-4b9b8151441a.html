REST API Demonstration

This demo shows how to interact with a REST API using JavaScript's Fetch API.

What is REST API?

REST (Representational State Transfer) is an architectural style for designing networked applications. A REST API uses HTTP requests to:

  • GET - Retrieve data
  • POST - Create data
  • PUT - Update data
  • DELETE - Delete data
