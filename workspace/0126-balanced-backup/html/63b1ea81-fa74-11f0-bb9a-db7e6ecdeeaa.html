PageRank Algorithm Demo

PageRank is an algorithm used by search engines to rank web pages in their search results. It works by counting the number and quality of links to a page to estimate how important the page is. The underlying assumption is that more important pages are likely to receive more links from other pages.

This demo shows a simplified network of 5 pages (A, B, C, D, E). Each page links to some others, and we iteratively calculate PageRank values for each page using the formula: 

    PR(A) = (1 - d)/N + d * ∑ [ PR(page_i) / L(page_i) ]

where:

  • d is the damping factor (usually ≈ 0.85)
  • N is total number of pages
  • L(page_i) is number of outbound links from page_i
  • The sum is over all pages linking to A
