Semaphore — Interactive Demo

A semaphore is a sync primitive that manages a set of permits. acquire() consumes a permit (or waits if none available), and release() returns a permit (possibly waking a waiting task). This demo simulates workers trying to acquire permits, run for a short time, then release them. Toggle fairness, spawn workers, and watch how permits and the wait queue change.
