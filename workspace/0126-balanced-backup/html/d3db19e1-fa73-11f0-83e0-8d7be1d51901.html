Explanation
Static typing (as in TypeScript) checks the code for type mismatches before you run it. That prevents many classes of bugs, like passing wrong argument types, accessing missing properties, or incorrectly assuming a function returns a certain type. This demo compiles TypeScript in your browser and shows diagnostics produced by the compiler.
This page uses the TypeScript compiler loaded in your browser. No code leaves your machine.
