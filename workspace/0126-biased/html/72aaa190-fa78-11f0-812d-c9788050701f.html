About Prim's Algorithm

Prim's algorithm is a greedy algorithm that finds a minimum spanning tree for a weighted undirected graph. This means it finds a subset of the edges that forms a tree that includes every vertex, where the total weight of all the edges in the tree is minimized.

The algorithm operates as follows:

  • Start with any vertex as the initial tree
  • At each step, add the cheapest edge that connects the tree to a vertex not yet in the tree
  • Repeat until all vertices are included in the tree

Time complexity: O(E log V) using binary heap and adjacency list representation, where V is the number of vertices and E is the number of edges.
