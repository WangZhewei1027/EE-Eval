Breadth‑First Search
Observe a wave-like exploration of a graph — level by level, shortest‑path first.

// BFS (high-level)
queue = [start]     visited[start]=true
while queue not empty:
    node = queue.pop_front()
    for each neighbor of node:
       if not visited: mark visited, push neighbor
Small observation: BFS discovers nodes in waves — nodes at distance d are discovered before any node at distance d+1.
