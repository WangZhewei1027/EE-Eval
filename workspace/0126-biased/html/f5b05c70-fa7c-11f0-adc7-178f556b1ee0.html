Priority Queue

What is a Priority Queue?

A priority queue is a data structure that follows the FIFO (First-In-First-Out) principle, where the highest-priority element is always at the front of the queue.

How does it work?

A priority queue is implemented as a binary heap, where the parent node is always greater than or equal to its child nodes.

Here's a step-by-step explanation:

  1. Insert an element into the queue.
  2. If the queue is empty, add the element to the end (root).
  3. Otherwise, compare the element with its parent node.
  4. If the element is smaller than its parent, move the element up the heap.
  5. Repeat steps 3-5 until the element is at the root.

What is the time complexity of priority queue operations?

Insert: O(log n)

Delete: O(log n)

Search: O(1)

Example Use Cases:

A job scheduling system can use a priority queue to schedule tasks based on their priority.

A network routing algorithm can use a priority queue to determine the shortest path to each node.

JavaScript Code:

    function priorityQueue() {
        let queue = [];
        function insert(element) {
            queue.push(element);
            queue.sort((a, b) => b - a);
        }
        function deleteElement() {
            if (queue.length > 0) {
                return queue.shift();
            } else {
                return null;
            }
        }
        function searchElement() {
            return queue[0];
        }
        return {
            insert,
            deleteElement,
            searchElement
        };
    }
    let pq = priorityQueue();
    pq.insert("Job 1");
    pq.insert("Job 2");
    pq.insert("Job 3");
    console.log(pq.searchElement()); // Output: Job 1
    pq.deleteElement();
    console.log(pq.searchElement()); // Output: Job 2
    pq.deleteElement();
    console.log(pq.searchElement()); // Output: Job 3
    pq.deleteElement();
    console.log(pq.searchElement()); // Output: null
