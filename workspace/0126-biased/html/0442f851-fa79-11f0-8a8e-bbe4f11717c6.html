PageRank

Imagine a web page as a graph with nodes and edges. Each node represents a page, and each edge represents a link between pages. The PageRank algorithm assigns a score to each node based on its centrality in the graph.

Here's how it works: for each node, the PageRank score is calculated as the sum of the scores of its neighbors divided by the number of neighbors. The node with the highest score is considered the most important page.
