DOMAIN NAME SYSTEM

Decoding the Internet’s Address Book

The Domain Name System (DNS) is a hierarchical and decentralized naming system for computers, services, or other resources connected to the internet or a private network. It translates human-friendly domain names like example.com into the numerical IP addresses needed for locating and identifying computer services and devices.
