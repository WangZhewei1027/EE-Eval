Stack — The elegant simplicity of LIFO
A stack is a linear data structure that follows the Last-In-First-Out discipline. Items are added to the "top" and removed from the "top". The visual at left illustrates depth, order, and motion — watch new elements land on top and depart from the same face.

Pseudo
LIFO
push(stack, item) { stack[top + 1] = item top = top + 1 } pop(stack) { item = stack[top] top = top - 1 return item }
