What is a Deque?

A Deque is a data structure that follows the Last-In-First-Out (LIFO) principle, where the last element added to the Deque is the first one to be removed.

It is a linear data structure that supports efficient insertion and removal of elements from both ends.

How Does a Deque Work?

A Deque consists of a single array or linked list. When an element is added to the Deque, it is inserted at the end of the array. When an element is removed from the Deque, it is removed from the beginning of the array.

Algorithmic Operations

Some common operations performed on a Deque include:

  • Insert an element at the beginning of the Deque
  • Insert an element at the end of the Deque
  • Remove an element from the beginning of the Deque
  • Remove an element from the end of the Deque

Implementation in JavaScript

Here is an example implementation of a Deque in JavaScript:

            var Deque = function() {
                this.items = [];
            };
            Deque.prototype.unshift = function(element) {
                this.items.unshift(element);
            };
            Deque.prototype.push = function(element) {
                this.items.push(element);
            };
            Deque.prototype.pop = function() {
                return this.items.pop();
            };
            Deque.prototype.front = function() {
                return this.items[0];
            };
            Deque.prototype.back = function() {
                return this.items[this.items.length - 1];
            };

Demonstration

Here is a simple demonstration of the Deque operations:

            var deque = new Deque();
            deque.push(1);
            deque.push(2);
            deque.push(3);
            console.log(deque.front()); // 1
            console.log(deque.back()); // 3
            deque.pop();
            console.log(deque.front()); // 2
            deque.pop();
            console.log(deque.front()); // 1
