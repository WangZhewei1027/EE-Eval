Notes / Tips
  • Use Generate Key, then Sign Message, then Verify Signature.
  • Try RSA-PSS with varying salt lengths. Try ECDSA on different curves.
  • Use HMAC to see deterministic MAC vs asymmetric non-deterministic signatures.
  • You can export keys as PEM or JWK, re-import them, and test verification across keys.
  • Record and replay sequences to explore state transitions.
