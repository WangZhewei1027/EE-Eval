Space complexity measures the extra memory an algorithm requires relative to input size. It excludes the input itself and highlights working memory, auxiliary structures, and recursion stacks.
Watch the right panel to see memory blocks appear for different Big-O classes. The animation cycles through scenarios automatically — click "Pause" to freeze.
Insight: O(1) uses a fixed amount of memory regardless of n — think a few variables or pointers.
Practical: Be mindful of additional buffers, copies, and recursion depth; they often drive up space complexity.
