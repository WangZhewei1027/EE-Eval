Interactive DNS Explorer

This tool lets you explore Domain Name System (DNS) concepts interactively. You can:

  • Resolve domains step-by-step, simulating the full DNS lookup process
  • Explore DNS record types with custom inputs
  • Manually edit or create DNS zones and records
  • Explore caching and TTL
  • Simulate DNS query failures and errors
  • Explore DNS delegation, recursion, iterative queries
  • Interact with a curated DNS Query Simulator, Resolver Config, and Zone Editor
