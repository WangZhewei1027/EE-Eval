Time Complexity

Learn how to measure the time taken by different algorithms and data structures

What is Time Complexity?

Time complexity is a measure of how long an algorithm takes to complete, usually expressed as a function of the input size.

Big O Notation

Big O notation is a way to describe the time complexity of an algorithm using a mathematical formula. There are several types of Big O notation, including:

  • O(1)
  • O(log n)
  • O(n)
  • O(n log n)
  • O(2^n)
