Floyd–Warshall — All Pairs Shortest Paths

A graceful algorithmic ballet: dynamic relaxation across a matrix of distances.

What you'll see

Floyd–Warshall iteratively picks an intermediate vertex k and attempts to improve each pair distance (i, j) by routing through k: if dist[i][k] + dist[k][j] < dist[i][j], the matrix updates. Watch the matrix animate, the chosen k glow along the timeline, and the best path between the pair being evaluated pulse on the graph.
