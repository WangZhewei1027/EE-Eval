Linear Search

Introduction to Linear Search

Linear Search is a simple searching algorithm used to find an element in an array or list. It works by iterating through the array one element at a time until it finds the element or reaches the end.

Here's a step-by-step explanation of the algorithm:

  1. Start from the beginning of the array.
  2. Compare the first element with the target element.
  3. If they match, return the index of the target element.
  4. If they don't match, move to the next element in the array.
