Quick Sort

Quick Sort is a divide-and-conquer algorithm that works by selecting a 'pivot' element from the array and partitioning the other elements into two sub-arrays, according to whether they are less than or greater than the pivot. The sub-arrays are then recursively sorted.

How it Works

Here's a step-by-step explanation of the Quick Sort algorithm:

  1. Choose a 'pivot' element from the array.
  2. Partition the array around the pivot, so that all elements less than the pivot are on the left, and all elements greater than the pivot are on the right.
  3. Recursively apply the Quick Sort algorithm to the sub-arrays on the left and right.

Algorithm Steps

Here are the key steps involved in the Quick Sort algorithm:

Step Code Description
1. Choose a 'pivot' element from the array. ```javascript function choosePivot(arr) { let pivot = arr[Math.floor(Math.random() * arr.length)]; return pivot; } ```
2. Partition the array around the pivot. ```javascript function partition(arr, pivot) { let left = 0; let right = arr.length - 1; while (left < right) { if (arr[left] < pivot) { // Swap arr[left] and arr[right] [arr[left], arr[right]] = [arr[right], arr[left]]; left++; } else { right--; } } return left; } ```
3. Recursively apply the Quick Sort algorithm to the sub-arrays on the left and right. ```javascript function quickSort(arr, low, high) { if (low < high) { let pivotIndex = partition(arr, arr[low]); quickSort(arr, low, pivotIndex - 1); quickSort(arr, pivotIndex + 1, high); } } ```

Example Use Case

Here's an example of how Quick Sort works on an array of numbers:

            let arr = [5, 2, 9, 1, 7, 3];
            let sortedArr = quickSort(arr, 0, arr.length - 1);
