Divide and Conquer

Divide and Conquer is a problem-solving approach that breaks down complex problems into smaller, more manageable parts.

It involves dividing the problem into smaller sub-problems, solving each sub-problem, and then combining the solutions to the sub-problems to solve the original problem.

Algorithms that use Divide and Conquer include Merge Sort, Quick Sort, and Binary Search.

Types of Divide and Conquer

  • Divide and Conquer: This is the most common type of Divide and Conquer, where the problem is divided into smaller sub-problems and solved recursively.
  • Bottom-Up: This type of Divide and Conquer starts with the smallest possible sub-problems and builds up to the original problem.
  • Top-Down: This type of Divide and Conquer starts with the largest possible sub-problems and breaks down to the original problem.

Merge Sort

Merge Sort is a divide and conquer algorithm that sorts an array of elements by dividing it into smaller sub-arrays, sorting each sub-array, and then merging the sorted sub-arrays.

Here is a step-by-step textual description of Merge Sort:

  1. Take the first element from the array and place it in its final position.
  2. Compare the first element with the rest of the array and place it in the correct position.
  3. Repeat steps 2 and 3 until all elements have been placed in their final positions.
  4. Repeat steps 2 and 3 until all sub-arrays have been sorted.
  5. Merge the sorted sub-arrays.

Merge Sort is an efficient algorithm with a time complexity of O(n log n) in all cases.

Example Use Case

Suppose we have a list of numbers and we want to sort them in ascending order.

We can use Merge Sort to sort the list by dividing it into smaller sub-lists, sorting each sub-list, and then merging the sorted sub-lists.

Here is an example implementation of Merge Sort in JavaScript:

            function mergeSort(arr) {
                if (arr.length <= 1) {
                    return arr;
                }
                const mid = Math.floor(arr.length / 2);
                const left = mergeSort(arr.slice(0, mid));
                const right = mergeSort(arr.slice(mid));
                return merge(left, right);
            }

            function merge(left, right) {
                const result = [];
                while (left.length > 0 && right.length > 0) {
                    if (left[0] <= right[0]) {
                        result.push(left.shift());
                    } else {
                        result.push(right.shift());
                    }
                }
                result.push(...left);
                result.push(...right);
                return result;
            }

Here is an example of how to use Merge Sort to sort a list of numbers:

            const numbers = [64, 34, 25, 12, 22, 11, 90];
            const sortedNumbers = mergeSort(numbers);
            console.log(sortedNumbers);
