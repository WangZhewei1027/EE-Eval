Interpolation Search Explanation

Interpolation search is an improved variant of binary search for uniformly distributed, sorted arrays. Instead of always dividing the search space in half, it estimates the position of the target value using linear interpolation formula:

pos = low + ((x - arr[low]) * (high - low)) / (arr[high] - arr[low])

This makes it an O(log log n) algorithm in the best case, though it can degrade to O(n) in worst cases.
