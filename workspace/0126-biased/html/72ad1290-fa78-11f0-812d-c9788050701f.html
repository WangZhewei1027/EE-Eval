The Cosmic Compiler

A visual journey through the transformation of code, from human-readable syntax to machine-executable instructions.

✏️ Source Code
function fibonacci(n) { if (n < 2) { return n; } return fibonacci(n - 1) + fibonacci(n - 2); } // Calculate 10th Fibonacci number console.log(fibonacci(10));

🔧 Intermediate Representation
define i32 @fibonacci(i32 %n) { entry: %cmp = icmp slt i32 %n, 2 br i1 %cmp, label %ret_n, label %recursive ret_n: ret i32 %n recursive: %n1 = sub i32 %n, 1 %fib1 = call i32 @fibonacci(i32 %n1) %n2 = sub i32 %n, 2 %fib2 = call i32 @fibonacci(i32 %n2) %result = add i32 %fib1, %fib2 ret i32 %result }
