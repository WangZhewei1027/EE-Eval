Recursion Explorer (Interactive)

Demonstrate and explore recursion with many interactive controls. Use templates or enter a custom ternary-style recursive expression. The expression must be of the form: condition ? baseReturn : recursiveReturn, and use recurse() inside the recursiveReturn. Parameter name: n. You may use Math.* and a branching variable b in generated templates.

Controls
Recursive Expression (ternary)

Format: <condition> ? <baseReturn> : <recursiveReturn> — use recurse(...) inside recursiveReturn

Interactive hints
  • Use Step to move one significant event at a time (entering a call, creating a child, or returning a call).
  • Auto Run will proceed at the delay above. Pause will stop it.
  • Back rewinds one snapshot (state before last event).
  • Build full recursion tree bypasses stepping and quickly constructs the whole tree, useful for large trees (respecting max depth).
  • Export/Import tree JSON to save and explore later.
