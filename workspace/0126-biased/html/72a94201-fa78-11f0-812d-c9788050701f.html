About Min Heaps

A Min-Heap is a binary tree where the value of each node is less than or equal to the values of its children. This property must be recursively true for all subtrees in the heap.

The minimum value is always at the root of the heap, making it efficient for priority queue implementations where we frequently need to access the minimum element.

Key operations include:

  • Insert: Adds a new element and restructures the heap to maintain the heap property
  • Extract Minimum: Removes and returns the smallest element, then restructures the heap
