What you are seeing

P: A single deterministic computation evolving in polynomial time. NP: A space of many candidate solutions whose correctness can be checked quickly.

Focus
Limited controls: Observe the visuals. The mathematical question "Is P equal to NP?" remains open.

Formal snapshot

P — the class of decision problems solvable by a deterministic Turing machine in polynomial time. Example: sorting, shortest paths.

NP — the problems for which a candidate solution (certificate) can be verified by a deterministic Turing machine in polynomial time. Example: SAT, Hamiltonian cycle.

The central open problem: Does P = NP? It is unknown whether every efficiently verifiable problem is also efficiently solvable.
