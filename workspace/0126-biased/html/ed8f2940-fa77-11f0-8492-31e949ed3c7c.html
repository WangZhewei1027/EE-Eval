ACID Properties

A Concept in Transaction Management

Atomicity

Ensures that all operations within a transaction are completed; if not, the transaction is aborted.

Consistency

Guarantees that a transaction will bring the database from one valid state to another.

Isolation

Allows transactions to operate independently without interference, ensuring correct results.

Durability

Once a transaction has been committed, it will remain so, even in the event of a system failure.
