Floyd-Warshall Algorithm

Floyd-Warshall Algorithm is an algorithm for finding the shortest path between all pairs of vertices in a weighted graph. It is a variation of Dijkstra's algorithm.

It was first proposed by Robert C. Floyd in 1957 and developed by Stephen Warshall in 1960.

Here is a step-by-step textual description of the Floyd-Warshall Algorithm:

  1. Input: A weighted graph represented as an adjacency matrix.
  2. Output: The shortest path between all pairs of vertices.
  3. Algorithm:
    1. For each vertex i (1 ≤ i ≤ n), find all possible next vertices j (1 ≤ j ≤ n) that can be reached from i.
    2. For each pair of vertices i and j, update the shortest distance from i to j if a shorter path is found.
