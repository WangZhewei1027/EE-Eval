B-Tree Index — Interactive Demo

Tree Configuration
Minimum degree t defines the B-Tree properties. Max keys per node = 2*t - 1.
Single Operations
Supports numeric and string keys. Use Step Mode to see algorithmic transitions.
Bulk & Random Operations


Random inserts generate numbers in range 0..999 (or strings 'sNN').
Range & Scan
Range and traversal results printed to the log area. Inclusive range.
Stepper / Playback
Use Step Mode on operations to generate an algorithmic trace you can step through. History is recorded to allow undo/redo.
Manual Node Operations & Editor


Manual operations can break invariants; use Validate button to check consistency. IDs are assigned internally.
Export / Import / Undo
Export copies the current structure as JSON to a text box. Import accepts JSON to replace tree. Undo/Redo available.
Diagnostics & Controls
Use validate to ensure basic B-Tree properties hold. Compact rebuilds tree by reinserting keys in-order.
