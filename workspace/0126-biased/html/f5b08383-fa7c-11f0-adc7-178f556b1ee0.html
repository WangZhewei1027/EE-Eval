Insertion Sort

Insertion Sort is a simple sorting algorithm that works by dividing the input into a sorted and an unsorted region. Each subsequent element from the unsorted region is inserted into the sorted region in the correct position.

Here is a step-by-step textual description of the Insertion Sort algorithm:

  1. Start with the first element of the input array as the sorted region.
  2. Compare the first element of the input array with the elements in the unsorted region.
  3. If the first element of the input array is less than or equal to the first element of the unsorted region, insert the first element into the sorted region.
  4. Move to the next element in the unsorted region and repeat the comparison and insertion steps.
  5. Continue this process until the entire input array is sorted.

Here is an educational narrative that illustrates the Insertion Sort algorithm:

Imagine you have a list of numbers from smallest to largest. You start by placing the smallest number in the correct position in the sorted region. Then, you compare the smallest number with the next number in the unsorted region. If it's smaller, you insert it into the sorted region. This process continues until the entire list is sorted.
