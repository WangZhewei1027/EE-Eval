Big-Omega Notation Definition

Big-Omega (Ω) notation describes the asymptotic lower bound of a function.

Formally: f(n) is Ω(g(n)) if there exist positive constants c and n₀ such that:

0 ≤ c·g(n) ≤ f(n) for all n ≥ n₀

Proof Technique

To prove f(n) is Ω(g(n)), we need to find constants c > 0 and n₀ ≥ 0 such that:

f(n) ≥ c·g(n) for all n ≥ n₀

Proof Strategies

  • Limit Method: Compute lim(n→∞) f(n)/g(n)
  • Inequality Construction: Find c where f(n) ≥ c·g(n)

Big-Omega Properties

Property Explanation
Reflexivity f(n) is Ω(f(n))
Transitivity If f(n) is Ω(g(n)) and g(n) is Ω(h(n)), then f(n) is Ω(h(n))
Not symmetric f(n) is Ω(g(n)) does not imply g(n) is Ω(f(n))
