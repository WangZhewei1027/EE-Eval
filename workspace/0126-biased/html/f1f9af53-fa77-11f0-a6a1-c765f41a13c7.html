Deterministic

Given the same input, a hash function always produces the same fixed-size output. The digest is a compact "fingerprint" of the message.

Avalanche Effect

Small differences in the input (even a single bit) cause large, unpredictable differences in the output. This is visible as a totally different color pattern for similar messages.

Fixed Output Length

No matter the input size, the digest length remains constant (SHA‑256 produces 256 bits, i.e., 32 bytes). The visualization maps each byte to a colored tile.
