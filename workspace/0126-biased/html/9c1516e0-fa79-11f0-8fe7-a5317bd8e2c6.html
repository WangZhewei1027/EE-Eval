Explanation & Notes

    
Select an algorithm and parameters, then Generate Events. Use Step / Play to see allocations and frees over time. Use the custom script for advanced patterns (allocate/free/ref). The memory listing shows live allocations and a simple textual size bar. The estimator computes numeric sizes for chosen complexity.
