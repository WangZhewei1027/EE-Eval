Bubble Sort Interactive (Minimal UI)

This page lets you explore Bubble Sort deeply. Edit data, generate traces, step forward/back in micro-steps (compare, swap), autoplay, set breakpoints, export/import states, and inspect a complete execution trace. Visuals are intentionally minimal.

Array Setup
After applying a new array you must "Recompute Trace" to update the instrumented steps. "Use Current" fills the text input with the currently displayed snapshot.

Explanation / Hints
- Trace entries are micro-steps: "compare" indicates a comparison of elements at j and j+1. If a swap is needed, a following "swap" step shows the array after swapping. If no swap, a "no-swap" step may appear if micro-step is enabled. Each pass_end marks the end of an outer loop pass.
- Use breakpoints to pause when interesting events occur.
- You can jump to any trace step by clicking it in the trace list. Use "Recompute Trace" after changing initial array or options.
