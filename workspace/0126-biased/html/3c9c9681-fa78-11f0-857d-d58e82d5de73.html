A hash function takes input data of arbitrary size and produces a unique, fixed-length hash output, also known as a message digest or fingerprint. This process is deterministic, so the same input always produces the same output, but a tiny change in input leads to a drastically different hash.

Hash functions are foundational in data integrity, cryptography, and computer science. They enable quick data comparisons, password storage, digital signatures, and blockchain security.
