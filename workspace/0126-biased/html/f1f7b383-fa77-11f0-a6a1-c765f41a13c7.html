What is NoSQL?

NoSQL is an umbrella of database technologies that depart from traditional table-based relational models. It prioritizes flexibility, horizontal scalability, and specialized data models — document stores, key-value stores, wide-column stores, and graph databases. These patterns let applications store complex, nested, and highly connected data with pragmatic performance characteristics for modern, distributed systems.
