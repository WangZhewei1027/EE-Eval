About Kruskal's Algorithm

Kruskal's algorithm is a greedy algorithm that finds a minimum spanning tree for a connected weighted graph. It works by sorting all the edges from the lowest weight to highest weight, then picking the smallest edge and checking if it forms a cycle with the spanning tree formed so far.

The algorithm continues until there are (V-1) edges in the spanning tree, where V is the number of vertices in the graph. The visualization above demonstrates this process step by step, highlighting the current edge being considered and showing which edges are added to the minimum spanning tree.
