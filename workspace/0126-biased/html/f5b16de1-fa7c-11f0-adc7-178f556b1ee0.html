Prim's Algorithm

What is Prim's Algorithm?

Prim's algorithm is a popular graph traversal algorithm used to find the minimum spanning tree of a connected, undirected, and weighted graph. It works by selecting a starting vertex and then exploring its neighbors, adding edges to the tree if they are not already included, and repeating this process until all vertices have been visited.

Theory

The algorithm starts by choosing an arbitrary vertex as the starting point. Then, it iteratively selects the vertex with the minimum number of edges connected to it and adds it to the tree. The algorithm continues until all vertices have been visited. The resulting tree is the minimum spanning tree of the original graph.

Algorithms

Prim's algorithm has a time complexity of O(E log V) in the worst case, where E is the number of edges and V is the number of vertices. This makes it efficient for large graphs. However, in the best case, the algorithm has a time complexity of O(E log E), which occurs when the graph is already a tree and we can simply add all edges to the tree.

Example

Consider a graph with 6 vertices and the following edges: (1, 2), (1, 3), (2, 4), (3, 5), (4, 6), and (5, 6). The minimum spanning tree of this graph is (1, 2), (1, 3), (2, 4), and (3, 5). The resulting graph has 4 edges and 3 vertices.

Code

function prim(graph, start) {
    let visited = new Set();
    let edges = graph.edges;

    // Choose an arbitrary vertex as the starting point
    let currentVertex = start;
    visited.add(currentVertex);

    let tree = [];

    while (edges.length > 0) {
        let edge = edges[0];
        edges.shift();

        // If the edge is already included in the tree, skip it
        if (visited.has(edge.vertex)) {
            continue;
        }

        // Add the edge to the tree and mark the vertex as visited
        tree.push(edge);
        visited.add(edge.vertex);

        // Select the vertex with the minimum number of edges connected to it
        let minVertex = null;
        let minEdges = Infinity;
        for (let vertex of visited) {
            let edgesConnectedToVertex = graph.edges.filter(edge => edge.vertex === vertex).length;
            if (edgesConnectedToVertex < minEdges) {
                minVertex = vertex;
                minEdges = edgesConnectedToVertex;
            }
        }

        // Add the vertex with the minimum number of edges connected to it to the tree
        currentVertex = minVertex;
    }

    return tree;
}

// Example usage
let graph = {
    'A': ['B', 'C'],
    'B': ['A', 'D', 'E'],
    'C': ['A', 'F'],
    'D': ['B'],
    'E': ['B', 'F'],
    'F': ['C', 'E']
};

let startVertex = 'A';
let tree = prim(graph, startVertex);

console.log('Minimum Spanning Tree:');
for (let edge of tree) {
    console.log(edge);
}
