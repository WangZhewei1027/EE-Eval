HTTP Explanation

The Hypertext Transfer Protocol (HTTP) is a request-response protocol used for transferring data over the internet.

It is a standardized communication protocol between a client and a server, allowing them to exchange data in a structured and format-specific way.

HTTP is typically used for transferring data in the following formats:

  • Text
  • Binary
  • JSON

HTTP uses the following components to make requests and receive responses:

  • Request
  • Response
  • Headers
  • Body

HTTP requests typically start with the following format:

        GET /path/to/resource HTTP/1.1

Where:

  • GET
  • /path/to/resource
  • HTTP/1.1

HTTP responses typically have the following format:

        HTTP/1.1 200 OK

Where:

  • HTTP/1.1
  • 200 OK

HTTP also uses the following status codes to indicate the outcome of a request:

  • 1xx
  • 2xx
  • 3xx
  • 4xx
  • 5xx

Here is an example of an HTTP request and response:

        
        HTTP/1.1 200 OK
HTTP/1.1 301 Moved Permanently

Where:

  • HTTP/1.1
  • 200 OK
  • 301 Moved Permanently
