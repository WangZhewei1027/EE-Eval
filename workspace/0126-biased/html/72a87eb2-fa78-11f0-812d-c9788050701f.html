How a Stack Works

A stack is an abstract data type that serves as a collection of elements, with two principal operations: push, which adds an element to the collection, and pop, which removes the most recently added element.

The order in which elements come off a stack gives rise to its alternative name, LIFO (last in, first out). The most recently added item is the first to be removed.
