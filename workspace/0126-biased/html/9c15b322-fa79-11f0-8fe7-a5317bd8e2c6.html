Query Optimization Explorer

This is a plain, interactive explorer for query optimization concepts. It emphasizes functional interactivity over visual polish. Use the controls below to build a query, adjust statistics, enable indexes, and try different optimizer algorithms (dynamic programming, greedy, randomized, genetic-like). You can step through the DP optimization process, view plan space, compare and simulate plans, and save/load state as JSON.

Tables & Statistics (add up to 6)

For each table specify estimated rows and whether the table is included in the query.

Join Predicates

Enable predicates between table pairs and set selectivities (0.0 - 1.0). Lower selectivity means fewer rows after join.

Indexes & Hints

Enable simple single-column index hints. Indexes reduce scan cost for that table for selective predicates.
