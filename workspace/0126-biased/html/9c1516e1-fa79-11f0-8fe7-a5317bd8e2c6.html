NP-Completeness Interactive Playground

This single-page app lets you create instances of common NP-complete problems (3-SAT, CLIQUE, VERTEX COVER, SUBSET-SUM), run multiple solvers (brute force, backtracking, greedy), perform classic reductions (3-SAT → CLIQUE, CLIQUE → VERTEX COVER), step through search procedures, and map solutions back. The interface is intentionally plain and unstyled — focus on exploring concepts and workflows.

1) Choose problem / instance type
Input format (one clause per line): space-separated integers representing literals. Example: 1 -2 3 means x1 OR (NOT x2) OR x3. Use variable indices 1..n.
Edge list format: one edge per line "u v" (1-based node indices). For an undirected graph include each edge once.
Optional CLIQUE target size:
Edge list format: one edge per line "u v" (1-based).
Optional vertex cover target size:
Format: numbers separated by spaces or newlines, and a target on the next line after an empty line or prefix "T=".
2) Reductions
After applying a reduction you can examine the target instance in its editor, run solvers on it, and then map found solutions back to the original problem with "reverse mapping".
