How this demo works

The visualization performs a depth-first exploration of a small search tree. Each node shows an upper bound (UB) — an optimistic estimate of the best value attainable below that node. When we find a solution (a leaf) with a value exceeding the current incumbent, we update the incumbent. Any subtree whose UB is not better than the incumbent can be safely pruned (skipped), saving exploration time.
