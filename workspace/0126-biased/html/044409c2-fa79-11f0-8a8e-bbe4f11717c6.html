Overview of TCP/IP

TCP/IP (Transmission Control Protocol/Internet Protocol) is a set of protocols used for communication over the internet.

TCP/IP is the most widely used protocol suite in the internet.

TCP (Transmission Control Protocol) is a connection-oriented protocol that ensures reliable data transfer.

IP (Internet Protocol) is a connectionless protocol that provides logical addressing.
