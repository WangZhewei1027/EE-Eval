B-Tree Index

Visual Concept & Structure

The B-Tree index is a balanced tree data structure widely used in databases to efficiently store and retrieve sorted data. It maintains sorted keys within nodes, allowing logarithmic search, insertion, and deletion. This visualization depicts a classic order-3 B-Tree (max 2 keys per node, 3 children), illustrating the hierarchy and balanced nature of the index.
