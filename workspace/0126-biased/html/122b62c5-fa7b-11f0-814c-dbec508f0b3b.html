Insertion Sort

How it works:

Insertion Sort is a simple sorting algorithm that works by iterating through a list of elements one by one. It compares each element with its adjacent elements and inserts it into its correct position in the sorted list.

How to use:

  • Enter a list of numbers separated by commas.
  • Press the "Sort" button to run the algorithm.
  • Press the "Reset" button to clear the list.
  • Press the "Help" button to view the algorithm's steps.
