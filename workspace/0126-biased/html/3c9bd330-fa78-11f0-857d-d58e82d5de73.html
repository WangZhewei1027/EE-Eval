Understanding Logistic Regression

Logistic Regression is a fundamental statistical model used for binary classification tasks. It predicts the probability that a given input belongs to a particular class using the logistic (sigmoid) function.

The sigmoid curve smoothly maps any real-valued number into the range (0,1), which can be interpreted as a probability:

σ(x) = 1 / (1 + e-(β0 + β1x))

Data points are plotted on this visualisation with two classes represented by distinct colors. The logistic curve models a smooth decision boundary separating these classes by probabilities.
