Sockets provide the endpoint abstraction for network communication. This illustration animates the lifecycle: initiating a connection (SYN), acknowledging (SYN-ACK / ACK), exchanging data packets, and performing a clean close (FIN).

// Pseudo socket lifecycle (in many languages)
socket = socket(domain, type, protocol);
socket.connect(address); // SYN → SYN-ACK → ACK
socket.send(data); // reliable, ordered stream
socket.close(); // FIN → ACK → FIN → ACK
