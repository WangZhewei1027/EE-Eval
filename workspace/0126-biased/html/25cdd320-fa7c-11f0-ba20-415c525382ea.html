Understanding Linear Regression: A Comprehensive Explanation

Linear Regression is one of the most fundamental algorithms in the field of statistics and machine learning used to model the relationship between a scalar dependent variable and one or more explanatory (independent) variables. In this page, we will explore the concept in detail — covering the core theory, mathematical foundations, and the intuition behind how it works.

1. What is Linear Regression?

At its core, Linear Regression is a method to predict a continuous outcome variable based on one or more predictor variables by fitting a linear relationship between them. The key assumption is that the relationship between the dependent variable (y) and independent variables (x) can be modeled as a straight line or a linear combination.

Simple Linear Regression: When there is only one predictor variable, the model assumes the form:

y = β0 + β1x + ε

Where:

  • y = dependent variable (outcome)
  • x = independent variable (predictor)
  • β0 = regression intercept (value of y when x=0)
  • β1 = regression slope (the change in y per unit change in x)
  • ε = error term (captures noise or randomness)

Multiple Linear Regression: For multiple predictors x₁, x₂, ..., xp, the model generalizes to:

y = β0 + β1x1 + β2x2 + ... + βpxp + ε

2. Why Use Linear Regression?

Some main uses and reasons for adopting linear regression include:

  • Prediction: Given new input data, provide quantitative predictions.
  • Interpretability: The coefficients (β) give a clear, direct understanding of variable influence.
  • Foundation method: Serves as a basis for more complex regression models.
  • Computationally efficient: Simple to compute and fast to execute on even large datasets.

3. The Goal of Linear Regression

The goal is to find the best-fitting line (or hyperplane, in multivariate regression) through the data points. "Best-fitting" formally means choosing the coefficients (β) such that the prediction errors are minimized.

Commonly, the Least Squares Criterion is used — minimizing the sum of squared differences between the observed values and predicted values:

minβ0, β1, ..., βp ∑i=1n (yi - ŷi)²

Where ŷi = β0 + β1xi1 + ... + βpxip is the predicted value for observation i.

4. How Does Linear Regression Work? Step-By-Step Explanation

Let's walk through the process conceptually:

  1. Collect Data: Record instances of the independent variable(s) x and the dependent variable y.
  2. Assume a Linear Model: Suppose the relationship between x and y can be represented as y = β0 + β1x + ε.
  3. Define a Loss Function: Typically, choose the sum of squared errors loss function: 
    Loss(β0, β1) = ∑ (yi - (β0 + β1xi))²
  4. Optimize Coefficients: Find the values of β0 and β1 which minimize the loss. This can be done through calculus by setting partial derivatives to zero: 
    ∂Loss/∂β0 = 0
∂Loss/∂β1 = 0
  5. Calculate the Estimates: The closed-form solutions are:
    β̂1 = ∑(xi - x̄)(yi - ȳ) / ∑(xi - x̄)²
    β̂0 = ȳ - β̂1x̄
    Where x̄ and ȳ are the sample means of x and y, respectively.
  6. Make Predictions: For a new input x*, predict 
    ŷ* = β̂0 + β̂1x*

5. Key Intuition

Imagine you have a scatter plot of points showing data pairs (xi, yi). Linear regression tries to draw a straight line that best captures the overall trend. The slope tells you how much y changes on average for a unit change in x. The intercept is where this line crosses the y-axis (the value of y when x=0).

The "best" line is the one that minimizes the vertical distances (errors) from the points to the line, squared to heavily penalize larger errors.

6. Assumptions of Linear Regression

For Linear Regression to produce trustworthy results, several assumptions are generally made:

  • Linearity: The relationship between predictors and outcome is linear.
  • Independence: Observations are independent of each other.
  • Homoscedasticity: The variance of error terms is constant across values of x.
  • Normality of Errors: Errors are normally distributed (important for inference).
  • No Multicollinearity (Multiple Regression): Predictor variables are not highly correlated.

7. Example: Predicting House Prices Using Size

Suppose you want to predict the price of a house (y) based on its size in square feet (x). You collect data on houses sold:

House Size (sq ft) (x) Price ($1000) (y)
750150
800160
850170
900180
950190

The goal is to find coefficients β0 and β1 that best fit this data. Computing the mean of x and y, variances, and covariances, then applying the formulas yields:

x̄ = 850, ȳ = 170
β̂1 = cov(x,y) / var(x) = 0.2
β̂0 = 170 - 0.2 × 850 = 0

So the regression line becomes:

Price = 0 + 0.2 × Size

If you wanted to estimate the price of a 1000 sq ft house:

Price = 0 + 0.2 × 1000 = $200,000

8. Linear Regression in Matrix Form (Multiple Predictors)

When there are multiple predictors, it is often convenient to use matrix notation for compactness and efficient computation. Suppose you have:

  • y — vector of observations (n×1)
  • X — matrix of predictor variables (n×(p+1)), where the first column is all ones for the intercept.
  • β — vector of coefficients ((p+1)×1)
  • ε — vector of error terms (n×1)

The linear regression model is written as:

y = Xβ + ε

The Least Squares solution that minimizes ||y - Xβ||² is given by the normal equation:

β̂ = (XTX)-1XTy

This formula provides the closed-form solution for β, assuming the matrix XTX is invertible.

9. Limitations and When to Use Caution

While Linear Regression is simple and powerful, it does not work well when:

  • The relationship between x and y is non-linear.
  • The errors are not normally distributed or have heteroscedasticity.
  • There are outliers that skew the regression line.
  • Predictors have multicollinearity issues.

In such cases, other models like polynomial regression, ridge/lasso regression, generalized linear models, or non-parametric methods may be more appropriate.
