SQL Concept Interactive Exploration

This interactive tool allows you to build, edit, and run simple SQL queries on example in-memory tables. Experiment with SQL concepts by:

  • Creating and modifying tables and their data
  • Writing SELECT queries with WHERE, JOIN, GROUP BY, HAVING, ORDER BY, LIMIT
  • Using aggregate functions (COUNT, SUM, AVG, MIN, MAX)
  • Seeing query results displayed as a table
  • Exploring SQL clauses through interactive controls
1. Define or Edit Table
2. Compose SQL Query (limited to SELECT statements)
3. Interactive Query Builder Controls
4. Tables Data
5. Query Results
