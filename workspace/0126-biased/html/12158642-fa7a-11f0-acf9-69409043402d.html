Deadlock Concept Interactive Simulator

This tool simulates a system with multiple processes and resources to explore deadlocks. You can create processes, define resource requests and allocations, simulate step-by-step execution, and experiment with different scheduling / resource granting policies to see deadlock conditions arise or get resolved.
