What you’re seeing

Counting Sort tallies how many times each value appears (count buckets), transforms those counts into cumulative positions, and then places each input element into the right index in the output array. This visualization highlights those stages: tallying, prefix-summing, and placing items — all while preserving stability.

Moving elements
Elements animate from Input → Count → Output with graceful motion and subtle emphasis.

Algorithm Summary

1. Count occurrences for each value.
2. Convert counts to cumulative positions.
3. Iterate input (reverse for stability), place each element using cumulative index, decrement count.
