Circular Linked List

A Circular Linked List (CLL) is a variation of the traditional linked list where the last node points back to the first node instead of pointing to null. This arrangement forms a circular structure, allowing traversal of the list starting from any node.

Key Concepts

Understanding Circular Linked Lists involves familiarizing oneself with several key terms and concepts:

  • Node: Each element in the list that contains data and a reference (or pointer) to the next node.
  • Head: The first node in the list. In a circular linked list, this node points to the next node in the sequence.
  • Tail: The last node in the list, which points back to the head, completing the circular connection.

Structure of a Circular Linked List

Visually, a circular linked list can be represented as follows:

        Head → Node1 → Node2 → Node3
                ↑               ↓
                <---------------

In this representation, if you start from Head and continue to follow the next pointers, you'll eventually loop back to Head.

Types of Circular Linked Lists

  • Single Circular Linked List: Each node points to its next node, and the last node points back to the first node.
  • Double Circular Linked List: Each node contains two pointers - one for the next node and one for the previous node, and the last node points back to the first node in both directions.

Advantages of Circular Linked Lists

Circular linked lists offer several advantages:

  • Continuous traversal: You can traverse the list starting from any node without needing to keep track of the head node.
  • Efficient: In scenarios where you need to rotate through the list frequently, circular linked lists can be efficient compared to linear counterparts.
  • Elimination of null values: CLLs remove the need for handling null references for the end of the list.

Common Operations

Insertion

Inserting a new node into a circular linked list can be done at the beginning, the end, or in between existing nodes. The basic steps for insertion can be illustrated as follows:

  1. To insert at the beginning, create a new node.
  2. Set the new node's next pointer to the head node.
  3. Set the last node's next pointer (tail) to the new node and update the head reference to the new node.

Deletion

Deleting a node in a circular linked list requires updating the pointers to remove the node from the sequence:

  1. Find the node that precedes the node to be deleted.
  2. Change the next pointer of the preceding node to skip over the deleted node.
  3. If the node to be deleted is the head, update the head reference.

Algorithm Example

Below is a simplified pseudocode for an algorithm to traverse a Circular Linked List:

        function traverseCLL(head):
            if head is None:
                return
            current = head
            do:
                print(current.data)
                current = current.next
            while current ≠ head
