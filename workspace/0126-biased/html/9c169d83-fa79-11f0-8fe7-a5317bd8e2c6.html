Logistic Regression Interactive Playground

Overview:
This interactive demo allows you to create or load 2D datasets, add/remove points manually, train logistic regression with many options (batch/SGD/Newton), step through training, explore decision boundary and probability surface, adjust regularization and learning rate, and evaluate metrics like loss, accuracy, confusion, and ROC/AUC. Use controls below. Visuals are intentionally minimal.
Canvas (click to add point).
Right-click a point to toggle its label. Shift+click to move last point.
