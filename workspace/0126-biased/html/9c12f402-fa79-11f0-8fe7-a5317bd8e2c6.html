Queue Playground

This interactive sandbox demonstrates queues (FIFO) with many controls. Everything is plain HTML. Use the controls to create queues, enqueue/dequeue, run automated producers/consumers, stage transactions, view history and logs, manage multiple queues, export/import state, and experiment with bounded/circular queues.
