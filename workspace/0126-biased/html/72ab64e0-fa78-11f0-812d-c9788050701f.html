NP-Completeness

A visual exploration of computational complexity and the fascinating boundary between tractable and intractable problems

P

Polynomial Time Problems

Problems that can be solved by a deterministic Turing machine in polynomial time (O(nk) for some constant k). These are considered "tractable" as their running time grows at a manageable rate with input size.

NP

Nondeterministic Polynomial Time

Problems whose solutions can be verified in polynomial time by a deterministic Turing machine. This class includes all P problems and potentially others where finding the solution might be harder than verifying it.

NPC

NP-Complete Problems

The most fascinating class in computational complexity - problems that are in NP and to which every other NP problem can be reduced in polynomial time. If any NP-complete problem has a polynomial-time solution, then P = NP.

P vs NP

The Millennium Problem

The question of whether P = NP is one of the most important unsolved problems in computer science, with a $1 million prize for its solution. Most researchers believe P ≠ NP, meaning that NP-complete problems are inherently difficult and no efficient algorithm exists for them.
