Relational Database

Relational Database Systems are designed to store and manage data in a structured way, using tables and relationships between them.

A relational database is a collection of interconnected tables, each with its own set of columns and rows. The relationships between tables are defined using foreign keys, which link tables together.

Key concepts in relational databases include:

  • Tables
  • Columns
  • Rows
  • Foreign Keys
  • Primary Keys
  • Indexing
  • Normalization

Relational Schema

A relational schema is the overall structure of a relational database, including the relationships between tables and the columns within them.

Common elements of a relational schema include:

  • Table name
  • Table type
  • Column name
  • Column data type
  • Column constraints

Relational Indexes

Relational indexes are used to improve query performance by providing an index of the columns used in a query.

Types of relational indexes include:

  • B-tree indexes
  • B-tree search trees
  • Hash indexes
  • Bloom filters

Relational Constraints

Relational constraints are used to enforce data integrity and prevent data inconsistencies.

Types of relational constraints include:

  • Primary keys
  • Foreign keys
  • Unique constraints
  • Primary key constraints

Relational Operations

Relational operations include:

  • INSERT
  • UPDATE
  • DELETE

Relational Joins

Relational joins are used to combine data from two or more tables based on a common column.

Types of relational joins include:

  • INNER JOIN
  • LEFT JOIN
  • RIGHT JOIN
  • FULL JOIN

Relational Subqueries

Relational subqueries are used to evaluate a query and return a result set that is used to answer another query.

Types of relational subqueries include:

  • SELECT... subquery

Relational Transactions

Relational transactions are used to manage the flow of data in a database, ensuring that data consistency and integrity are maintained.

Types of relational transactions include:

  • READ COMMITTED
  • READ UNCOMMITTED
  • REPELLED
  • SNAP SHUT
