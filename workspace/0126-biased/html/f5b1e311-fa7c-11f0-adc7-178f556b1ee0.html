Big-Theta Notation

Big-Theta Notation is a mathematical notation used to describe the time or space complexity of an algorithm. It is a way to express the relationship between the time or space complexity of an algorithm and the size of the input.

It is denoted by Θ(𝑥), which means "Theta of 𝑥".

Big-Theta Notation is used to describe the upper bound of an algorithm's complexity. It is the best possible time or space complexity that can be achieved by an algorithm.

    function countElements(arr) {

            return 0;
        } else {

        }
    }
