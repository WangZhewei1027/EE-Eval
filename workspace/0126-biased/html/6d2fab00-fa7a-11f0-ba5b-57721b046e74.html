Big-Theta (Θ) Notation Explorer

Concept Explanation

Big-Theta (Θ) describes an asymptotic tight bound on the growth rate of a function. A function f(n) is Θ(g(n)) if there exist positive constants c₁, c₂, and n₀ such that:

0 ≤ c₁·g(n) ≤ f(n) ≤ c₂·g(n) for all n ≥ n₀

This means f(n) grows at the same rate as g(n) within constant factors.
