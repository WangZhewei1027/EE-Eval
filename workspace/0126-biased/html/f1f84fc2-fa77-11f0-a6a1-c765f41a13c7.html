Watch the window adjust when congestion occurs.

This aesthetic simulation illustrates Additive Increase / Multiplicative Decrease (AIMD): the sender slowly raises its congestion window until packet loss is detected (packets dropped at the router), at which point it reduces the window substantially and then increases again.

  • Minimal controls: Start / Pause and Reset.
  • Observe queue buildup, packet drops, window (cwnd) changes and throughput.
