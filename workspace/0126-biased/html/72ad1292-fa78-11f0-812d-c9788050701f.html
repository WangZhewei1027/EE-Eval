What is an Abstract Syntax Tree?

An Abstract Syntax Tree (AST) is a tree representation of the abstract syntactic structure of source code written in a programming language. Each node of the tree denotes a construct occurring in the source code.

ASTs are widely used in compilers, static analyzers, and development tools to represent the structure of program code. They omit syntactic details like parentheses and focus on the hierarchical relationships between parts of the code.

function factorial(n) {
    if (n <= 1) {
        return 1;
    }
    return n * factorial(n - 1);
}
