Stack

The stack is a data structure that follows the Last-In-First-Out (LIFO) principle, meaning the last element added to the stack will be the first one to be removed.

Here's a step-by-step explanation of how a stack works:

  1. Elements are added to the top of the stack using the `` method.
  2. Elements are removed from the top of the stack using the `` method.
  3. The element that was removed is then returned to its original position at the top of the stack.
  4. The top element is then removed again, and this process continues until the stack is empty.

Stack Operations

A stack has the following operations:

  • push(element)
  • pop(element)
  • peek(element)
  • isEmpty
