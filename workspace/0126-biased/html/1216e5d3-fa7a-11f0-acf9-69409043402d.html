Help & Instructions

  • Enter JavaScript code above and click "Parse Code" to generate the AST.
  • Use "AST View Mode" and "Initial Expand Level" to control display details.
  • Filter displayed nodes by typing in the filter box and pressing "Highlight Filter Matches".
  • Click on node summaries in the tree to select nodes and inspect/edit properties below.
  • After editing, apply changes or undo edits. Serialize and download the AST anytime.
  • The AST node tree uses <details> elements for expand/collapse. Keys and values can be inspected.
  • The interface supports deep interaction paths with node filtering, selection, editing, and serialization.
