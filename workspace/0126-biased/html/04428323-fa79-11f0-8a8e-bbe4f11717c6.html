Exponential Search

This is an example of exponential search. It uses a binary search algorithm to find a value in an array.

Imagine you have a huge library with millions of books. You want to find a specific book. You start by searching the first shelf, then the second shelf, and so on. You keep narrowing down the search area until you find the book.

Exponential search is similar. It starts with a small search area, then expands it to find the target value. The more you search, the smaller the search area gets.
