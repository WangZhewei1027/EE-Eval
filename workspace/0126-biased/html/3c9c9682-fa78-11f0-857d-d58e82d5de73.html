Original Message

The document or message you want to send or prove authenticity of.

Digital Signature

A cryptographic signature uniquely derived from message hash and sender's private key.

Message Hash

A unique, fixed-length digest produced by hashing the original message.

A Digital Signature ensures message authenticity and integrity by combining a cryptographic hash of the original message with the sender's private key. This signature can be verified by recipients using the sender's public key, confirming the data was unaltered and truly from the sender. Notice how the original message is hashed into a unique digest (represented by the circular hash icon), and then signed cryptographically (illustrated by the elegant signature stroke), binding the identity and the message securely.
