How A* Search Works

A* is an informed search algorithm that finds the shortest path between two points while avoiding obstacles. It combines the advantages of Dijkstra's algorithm (which finds the shortest path) and Greedy Best-First Search (which is fast but not optimal).

The algorithm uses a heuristic function to estimate the cost from the current node to the goal. At each step, it prioritizes nodes that minimize the total estimated cost (f(n) = g(n) + h(n)), where g(n) is the actual cost from the start node to n, and h(n) is the heuristic estimate from n to the goal.

When the heuristic function is admissible (never overestimates the true cost), A* is guaranteed to find the optimal path.
