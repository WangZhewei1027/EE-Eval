About Min Heap and Controls

This interactive demo lets you explore a min binary heap. The heap invariant is that each parent node is less than or equal to its children.

  • Build Heap: Inputs an array, turns it into a valid min heap.
  • Insert: Adds a new value maintaining the heap property.
  • Extract Min: Removes and returns the smallest element (the root).
  • Peek Min: Shows the smallest value without removing it.
  • Bulk Insert: Insert many random values quickly.
  • Serialize/Deserialize: Save/load heap state as a JSON array.
  • Step-by-Step Heapify: Visualize heapify operations from a specific node. You can step "down" or "up".
