Circular Linked List

Circular Linked List is a data structure where each node points to the next node, forming a circle. This structure is used to implement a doubly linked list, which allows for efficient insertion and deletion of elements at any position.

Key Characteristics:

  • Each node contains a data value and a reference (or link) to the next node.
  • Each node has a reference (or link) to the previous node.
  • The first and last nodes are special, as they do not point to any other node.

Algorithm:

  1. Insert a new node at the beginning of the list.
  2. Update the references of the new node to point to the current first node.
  3. Update the references of the new node to point to the new first node.
  4. Repeat steps 2-3 until the new node is inserted at the end of the list.

Implementation:

        
            function Node(data) {
                this.data = data;
                this.next = null;
                this.prev = null;
            }

            function CircularLinkedList() {
                this.head = null;
                this.tail = null;
                this.size = 0;
            }

            function insertAtHead(data) {
                const newNode = new Node(data);
                if (!this.head) {
                    this.head = newNode;
                    this.tail = newNode;
                } else {
                    newNode.next = this.head;
                    this.head.prev = newNode;
                    this.head = newNode;
                }
                this.size++;
            }

            function insertAtTail(data) {
                const newNode = new Node(data);
                if (!this.tail) {
                    this.head = newNode;
                    this.tail = newNode;
                } else {
                    newNode.prev = this.tail;
                    this.tail.next = newNode;
                    this.tail = newNode;
                }
                this.size++;
            }

            function printList() {
                const elements = [];
                let current = this.head;
                while (current) {
                    elements.push(current.data);
                    current = current.next;
                }
                return elements;
            }

Example Use Case:

Suppose we have a circular linked list with 5 nodes, where each node contains a value from 1 to 5.

Insert the values 1, 2, 3, 4, and 5 at the beginning and end of the list.

Output:

        
            [1, 2, 3, 4, 5]

Implementation Details:

Circular Linked List is implemented using a Node class, which contains a reference (or link) to the next node and the previous node.

The list is implemented using three pointers: the head, tail, and current node.

The head pointer points to the first node, the tail pointer points to the last node, and the current node pointer moves one step at a time through the list.

The insertAtHead and insertAtTail methods update the references of the new node to point to the current head and tail nodes, respectively.

The printList method traverses the list and returns an array of values.
