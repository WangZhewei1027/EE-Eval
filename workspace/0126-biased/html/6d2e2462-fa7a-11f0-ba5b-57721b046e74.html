Binary Search Explanation

Binary search is an efficient algorithm for finding an item in a sorted list. It works by repeatedly dividing the search interval in half.

Steps:

  1. Compare the target value to the middle element of the array
  2. If the target equals the middle element, return its position
  3. If the target is less than the search continues in the lower half
  4. If the target is greater, the search continues in the upper half
  5. Repeat until the target is found or the interval is empty

Time Complexity: O(log n)
