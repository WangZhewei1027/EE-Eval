Bellman-Ford Algorithm Interactive Demo

The Bellman-Ford Algorithm finds shortest paths from a source vertex to all other vertices in a weighted graph. It can detect negative weight cycles reachable from the source. This demo allows you to: - Build your own directed weighted graph (with negative weights allowed) - Input the source vertex - Step through the algorithm iteration by iteration or run it fully - See the distance table updates and edge relaxation details - Detect negative cycles interactively - Reset, clear, or import/export graph + settings Use the controls below to interact.
