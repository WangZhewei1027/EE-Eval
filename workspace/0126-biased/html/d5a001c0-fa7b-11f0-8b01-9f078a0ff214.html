Doubly Linked List

A Doubly Linked List is a type of linked list in which each node contains a data field and two pointers. One pointer links to the next node in the sequence and the other links to the previous node. This structure allows traversal in both directions: forward and backward.

Structure of a Doubly Linked List

Each node in a doubly linked list comprises three components:

  • Data: The value or information the node holds.
  • Next Pointer: A pointer pointing to the next node in the list.
  • Previous Pointer: A pointer pointing to the previous node in the list.

Diagram Representation

        [Prev] <-> [Data] <-> [Next]

In this representation, [Prev] points to the previous node, [Data] is the value stored, and [Next] points to the subsequent node.

Advantages of a Doubly Linked List

  • Bidirectional Traversal: You can navigate forwards and backwards through the list.
  • Efficient Deletion: If you have access to a node, you can delete it without traversing from the head of the list.
  • Flexibility: Easier to implement arbitrary deletion and insertion operations compared to singly linked lists.

Disadvantages of a Doubly Linked List

  • Increased Memory Usage: Each node requires extra memory for the previous pointer.
  • Complexity: Operations like insertion and deletion are more complex due to additional pointers to manage.

Operations on Doubly Linked Lists

1. Insertion

Inserting a new node can be done at the beginning, end, or in between nodes:

  1. To insert at the beginning:
    • Create a new node.
    • Point the new node’s next pointer to the current head.
    • Adjust the previous pointer of the current head to point to the new node.
    • Update the head to the new node.
  2. To insert at the end, you would traverse until the last node and adjust pointers accordingly.
  3. To insert in between, you need to traverse to the desired position, then adjust pointers of the surrounding nodes.

2. Deletion

Deleting nodes from a doubly linked list also varies:

  1. To delete the head:
    • Update the head to point to its next node.
    • Set the new head's previous pointer to null.
  2. For nodes in between or at the end, adjust the next pointer of the previous node and the previous pointer of the next node accordingly.

3. Searching

To search for a particular value, begin at the head and traverse through the list using the next pointers until the desired value is found or the end is reached.

Example Code (Python)

class Node:
    def __init__(self, data):
        self.data = data
        self.next = None
        self.prev = None

class DoublyLinkedList:
    def __init__(self):
        self.head = None

    def append(self, data):
        new_node = Node(data)
        if not self.head:
            self.head = new_node
            return
        last = self.head
        while last.next:
            last = last.next
        last.next = new_node
        new_node.prev = last

Conclusion

Doubly linked lists provide a versatile structure that enables efficient insertion and deletion operations. They are particularly useful in applications needing bidirectional traversal of data. Understanding how to implement and work with them is essential for effective data management in programming.
