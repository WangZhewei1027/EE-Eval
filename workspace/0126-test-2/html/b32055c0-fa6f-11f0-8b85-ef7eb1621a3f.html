Bubble Sort is a simple comparison-based sorting algorithm. It works by repeatedly stepping through the list to be sorted, comparing adjacent elements, and swapping them if they are in the wrong order. The process is repeated until the list is sorted. ### How Bubble Sort Works: 1. **Compare**: Start at the beginning of the list and compare the first two adjacent elements. 2. **Swap**: If the first element is greater than the second element, swap them. 3. **Repeat**: Move to the next pair of adjacent elements and repeat the compare-and-swap process. 4. **Pass**: After each full pass through the list, the largest unsorted element will have bubbled up to its correct position at the end of the list. 5. **End Condition**: The algorithm is complete when a pass through the list is made without any swaps, indicating that the list is sorted. ### Time Complexity: - **Best Case**: O(n) when the list is already sorted. - **Average Case**: O(n^2) - **Worst Case**: O(n^2) ### Space Complexity: - **O(1)** (in-place sorting) ### Example of Bubble Sort in Python: ```python def bubble_sort(arr): n = len(arr) for i in range(n): # A flag to detect if any swaps were made swapped = False for j in range(0, n - i - 1): if arr[j] > arr[j + 1]: # Swap if the element found is greater than the next element arr[j], arr[j + 1] = arr[j + 1], arr[j] swapped = True # If no swaps were made, the array is sorted if not swapped: break # Example usage arr = [64, 34, 25, 12, 22, 11, 90] print("Original array:", arr) bubble_sort(arr) print("Sorted array:", arr) ``` ### Notes: - Bubble Sort is not suitable for large datasets as its average and worst-case time complexity is quite high. - It is mainly used for educational purposes to illustrate basic sorting algorithms.
