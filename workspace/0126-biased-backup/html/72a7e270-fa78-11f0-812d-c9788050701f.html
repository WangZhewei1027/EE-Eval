Array Fundamentals

An array is a systematic arrangement of similar objects, typically in rows and columns. In computer science, arrays are among the oldest and most important data structures.

Each element in an array can be accessed by its numerical index. The illustration above shows a simple array with five elements, each containing a unique value.

Arrays are zero-indexed in most programming languages, meaning the first element is at position 0, not 1.
