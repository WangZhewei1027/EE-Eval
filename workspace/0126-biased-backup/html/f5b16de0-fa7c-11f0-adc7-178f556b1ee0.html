Kruskal's Algorithm

Kruskal's Algorithm is a popular graph traversal algorithm used to find the minimum spanning tree of a connected, undirected, and weighted graph. It was first introduced by Harold Kruskal in 1956.

Detailed Written Explanation

Kruskal's Algorithm works by selecting the smallest edge that does not form a cycle in the graph. It starts by sorting the edges in non-decreasing order of their weights. Then, it iterates through the sorted edges and includes each edge in the minimum spanning tree if it does not form a cycle. The algorithm continues until all vertices are connected.

In the worst-case scenario, the algorithm may have to select all edges, resulting in a tree with n-1 edges. However, in the best-case scenario, it may only select one edge, resulting in a tree with n edges.

Step-by-Step Textual Descriptions

  1. Sort the edges in non-decreasing order of their weights.
  2. Initialize an empty minimum spanning tree.
  3. Iterate through the sorted edges and include each edge in the minimum spanning tree if it does not form a cycle.
  4. Continue until all vertices are connected.

Algorithmic Complexity

Kruskal's Algorithm has a time complexity of O(E log E) in the worst-case scenario and O(E log E) in the best-case scenario.

It has a space complexity of O(E) due to the sorting of edges.

Code Implementation

                function kruskal(graph) {
                    // Sort the edges in non-decreasing order of their weights
                    graph.edges.sort((a, b) => a.weight - b.weight);

                    // Initialize an empty minimum spanning tree
                    let mst = [];

                    // Iterate through the sorted edges and include each edge in the minimum spanning tree if it does not form a cycle
                    for (let edge of graph.edges) {
                        if (!includesCycle(graph, edge, mst)) {
                            mst.push(edge);
                        }
                    }

                    return mst;
                }

                function includesCycle(graph, edge, mst) {
                    // Perform a depth-first search to detect cycles
                    function dfs(currentNode, visited, path) {
                        if (visited[currentNode]) {
                            return false;
                        }

                        visited[currentNode] = true;
                        path.push(currentNode);

                        for (let neighbor of graph.edges) {
                            if (neighbor.weight === edge.weight && dfs(neighbor.node, visited, path)) {
                                return true;
                            }
                        }

                        path.pop();
                        return false;
                    }

                    return dfs(edge.node, {}, []);
                }
