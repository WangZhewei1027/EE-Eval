Hash Map Interactive Explorer

This tool demonstrates a hash map with many interactive controls. You may choose hash functions, collision strategies (separate chaining, linear/quadratic probing, double hashing), insert/delete/search/update keys, step through probe sequences, rehash, undo/redo, import/export state, edit a custom hash function, and explore internal details like buckets and probe statistics. The interface is intentionally plain and functional.
