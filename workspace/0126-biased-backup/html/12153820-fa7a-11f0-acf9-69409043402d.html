Big-Omega Notation Interactive Explorer

Big-Omega notation, Ω(g(n)), describes the asymptotic lower bound of a function f(n). Formally, f(n) = Ω(g(n)) if there exist constants c > 0 and n0 such that for all n ≥ n0, f(n) ≥ c·g(n).
