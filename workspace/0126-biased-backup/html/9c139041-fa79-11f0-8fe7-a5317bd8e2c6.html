Heap (Min) Interactive Explorer

This tool demonstrates a binary heap (default: Min Heap) with extensive interactive controls. You can insert values, extract the minimum, build the heap step-by-step (heapify), run heap sort, change keys, delete elements, undo/redo, play steps automatically, and inspect array or tree views. All visualization is plain text and minimal by design.
