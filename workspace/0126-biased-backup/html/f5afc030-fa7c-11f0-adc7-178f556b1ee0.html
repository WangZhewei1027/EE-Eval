Hash Table

Introduction

A hash table is a data structure that stores key-value pairs in a hash function, which maps keys to indices of an array. This allows for fast lookup, insertion, and deletion of elements.

Key-Value Pairs

Hash tables store key-value pairs in an array of buckets, where each bucket is associated with a hash value. The key is used to index the bucket, allowing for fast lookup and insertion.

For example, consider a hash table with 5 buckets, each containing 10 elements. The keys are "apple", "banana", "cherry", "date", and "elderberry". The corresponding values are stored in the buckets as follows:

Key Value
apple fruit
banana fruit
cherry fruit
date fruit
elderberry fruit

Hash Function

A hash function is a mathematical function that takes a key as input and returns a hash value, which is used to index the bucket.

There are many different hash functions, but one common approach is to use the modulus operator (%). The hash value is calculated as follows:

Hash = (key mod 10) % 10

This means that the key is divided by 10 and the remainder is taken as the hash value.

Insertion and Lookup

Insertion: To insert a new key-value pair into the hash table, we use the hash function to index the bucket. We then store the key-value pair in the corresponding bucket.

Lookup: To retrieve a value from the hash table, we use the hash function to index the bucket. We then search for the corresponding key in the array of buckets and return the associated value.

Example:

Suppose we have a hash table with 5 buckets, each containing 10 elements. We insert the following key-value pairs:

Key Value
apple fruit
banana fruit
cherry fruit
date fruit
elderberry fruit

To lookup the value of "apple", we use the hash function to index the bucket with key "apple". The hash value is calculated as follows:

Hash = (apple mod 10) % 10 = 2

We then search for the corresponding key in the array of buckets and return the associated value.

Result:

apple
