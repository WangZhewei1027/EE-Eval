How to read this visualization

Each node shows its current shortest-known distance from the source (A). The system performs N-1 relaxation passes across all directed edges in sequence. When an edge is highlighted, the algorithm attempts to relax it: if dist[u] + w < dist[v], the distance of v updates and it pulses.

At the end, the algorithm checks for negative cycles. If any distance can still be improved, that indicates the presence of a negative-weight cycle reachable from the source.
