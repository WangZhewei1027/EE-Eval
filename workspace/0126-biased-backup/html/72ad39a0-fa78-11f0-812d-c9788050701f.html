Type System

A visual exploration of how programming languages classify and manipulate data through static and dynamic typing paradigms.

Static Typing

Types are checked at compile-time. Variables must be declared with their type before use, and type mismatches are caught early in development.

Example (TypeScript) let message: string = "Hello";
message = 42; // Error: Type 'number' is not assignable to type 'string'
String
Number
Boolean

Dynamic Typing

Types are checked at runtime. Variables can hold values of any type, and type checking occurs as the program executes.

Example (Python) message = "Hello" # This works
message = 42 # This also works
Any
42
"Hello"
true
[1,2,3]

Type Inference

The compiler automatically deduces the type of an expression, reducing the need for explicit type annotations while maintaining type safety.

Example (Swift) let message = "Hello" // Compiler infers String
let count = 42 // Compiler infers Int
"Hello"
42
String
Int
