Help / Notes
This simulator demonstrates amortized analysis for dynamic arrays (push/pop) under different resizing policies. You can run sequences, step through operations, save snapshots, fork strategies, and compare costs. Methods shown: - Aggregate: total cost / operations. - Accounting: assign charges per op, track saved credits. - Potential: use a potential function phi; amortized = actual + phi_after - phi_before. Configure parameters per strategy and watch costs, potentials, and credits change.
