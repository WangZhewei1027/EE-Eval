REST API

Representational State Transfer — a lightweight architectural style for communication between clients and servers using stateless HTTP and resource-oriented interactions.

R

Resources

Everything is a resource identified by URIs — users, images, orders. Clients interact with representations of these resources via standard endpoints.

S

Stateless

Each request from client to server contains all information needed. The server does not rely on prior interactions, enabling scalability and simplicity.

V

HTTP Verbs

Use uniform interface: GET to retrieve, POST to create, PUT/PATCH to update, DELETE to remove. Semantics give APIs clarity and predictability.

Payloads & Representations
APIs transfer representations — JSON is common for its simplicity. Responses include status codes and standardized headers to convey state and meta information.
