Interpolation Search

Interpolation Search is a search algorithm that works by estimating the position of the target value within the array. It is a variation of Binary Search.

It is used when the array is not sorted and the target value is not known to be in the first or last element of the array.

Interpolation Search works by choosing a value between the first and last elements of the array and searching for the target value within that range.

It is a relatively simple algorithm to implement and is known for its high time complexity of O(log log n) on average.

Theory

Interpolation Search works by estimating the position of the target value within the array. It uses a range of possible positions and searches for the target value within that range.

Here is a step-by-step textual description of the algorithm:

  1. Choose a value k such that it is within the range [l, r] where l is the left endpoint and r is the right endpoint of the range.
  2. Calculate the estimated position of the target value using the formula: k = l + ((r - l) / (2 - m))
  3. Search for the target value within the range [l, k] using a binary search.
  4. Repeat the process until the target value is found or the range is empty.

Implementation

Here is an example implementation of the algorithm in JavaScript:

    function interpolationSearch(arr, target) {
        let l = 0;
        let r = arr.length - 1;
        let m;

        while (l <= r) {
            m = Math.floor((l + r) / 2);

            if (arr[m] === target) {
                return m;
            }

            if (arr[m] < target) {
                l = m + 1;
            } else {
                r = m - 1;
            }
        }

        return -1;
    }

    // Example usage
    let arr = [2, 5, 8, 12, 16, 23, 38, 56, 72, 91];
    let target = 23;

    let index = interpolationSearch(arr, target);

    if (index !== -1) {
        console.log("Target value found at index", index);
    } else {
        console.log("Target value not found in array");
    }
