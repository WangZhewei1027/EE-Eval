B-Tree Index — Visual Concept

A B-Tree stores sorted keys in balanced nodes to allow fast searches, inserts and deletes. Each internal node routes queries downward by comparing keys; leaves hold actual records or pointers. This animated illustration highlights a simple traversal searching for key 50.
