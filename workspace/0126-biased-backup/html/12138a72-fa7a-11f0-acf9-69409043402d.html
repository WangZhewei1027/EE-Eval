Instructions & Tips
  • Click on canvas background to add nodes (same as Add Node button).
  • Node names must be unique, max 10 chars, case sensitive.
  • To move nodes, toggle "Move Node Mode" and drag nodes on canvas.
  • Select a node by clicking it to delete it or use in edges.
  • Edges are undirected and defined by two existing nodes.
  • Highlighted nodes/edges indicate traversal path or cycle detection.
