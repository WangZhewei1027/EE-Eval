Doubly Linked List

Doubly Linked List (DLL) is a data structure that consists of a collection of nodes, where each node contains a value and a reference to the previous and next node in the sequence.

A DLL is similar to a singly linked list, but each node has two references: one to the previous node and one to the next node.

Key characteristics of DLLs are:

  • Each node contains a value and a reference to the previous and next node.
  • There are two references for each node: one to the previous node and one to the next node.
  • Nodes can be added or removed from the DLL without affecting the sequence of the DLL.

Implementation

Doubly Linked List Implementation:

Here is a simple implementation of a doubly linked list in JavaScript:

    class Node {
        constructor(value) {
            this.value = value;
            this.prev = null;
            this.next = null;
        }
    }

    class DoublyLinkedList {
        constructor() {
            this.head = null;
            this.tail = null;
        }

        // Add a new node to the DLL
        addNode(value) {
            const newNode = new Node(value);
            if (this.head === null) {
                this.head = newNode;
                this.tail = newNode;
            } else {
                newNode.next = this.tail;
                this.tail.prev = newNode;
                this.tail = newNode;
            }
        }

        // Remove the first node from the DLL
        removeHead() {
            if (this.head === null) {
                return null;
            }
            const removedNode = this.head;
            this.head = this.head.next;
            if (this.head === null) {
                this.tail = null;
            }
            return removedNode;
        }

        // Print the DLL
        printList() {
            let currentNode = this.head;
            while (currentNode !== null) {
                console.log(currentNode.value);
                currentNode = currentNode.next;
            }
        }
    }

    // Create a new DLL and add some nodes
    const dll = new DoublyLinkedList();
    dll.addNode(1);
    dll.addNode(2);
    dll.addNode(3);
    dll.addNode(4);

    // Print the DLL
    console.log("Printing DLL:");
    dll.printList();

    // Remove the first node
    const removedNode = dll.removeHead();
    console.log("Removed node:", removedNode.value);

    // Print the DLL again
    console.log("Printing DLL after removal:");
    dll.printList();

Theory

Doubly Linked List Theory:

Doubly Linked Lists are similar to singly linked lists, but each node has two references: one to the previous node and one to the next node.

Key aspects of doubly linked lists include:

  • Each node contains a value and a reference to the previous and next node.
  • There are two references for each node: one to the previous node and one to the next node.
  • Nodes can be added or removed from the DLL without affecting the sequence of the DLL.
  • Doubly linked lists are often used in applications where data needs to be efficiently traversed, such as in databases or file systems.

Applications

Doubly Linked List Applications:

Doubly linked lists have a wide range of applications, including:

  • Databases and file systems: Doubly linked lists are often used to efficiently traverse and manage large amounts of data.
  • Graph algorithms: Doubly linked lists are used in graph algorithms, such as depth-first search and breadth-first search.
  • Web browsers: Web browsers often use doubly linked lists to efficiently manage the navigation history of the user.
