The Two Pointers Technique

The Two Pointers technique is an efficient way to solve problems on sorted arrays, linked lists, or any iterable structures. This technique involves using two pointers (or indices) to traverse through the data structure, and it can help reduce the overall time complexity of the algorithm.

Concept Overview

The basic idea of the Two Pointers technique is to maintain two pointers that can move in different directions through the data structure. This can be useful for a variety of problems such as:

  • Finding pairs that sum up to a specific value.
  • Reversing a string or list in place.
  • Removing duplicates from a sorted array.
  • Searching for a subarray with a specific property.

Types of Two Pointer Techniques

While there are many applications of the Two Pointers technique, they can largely be classified into two main types:

1. The Left and Right Pointers

This method involves positioning one pointer at the start (left) and the other at the end (right) of the array. You can adjust the position of the pointers based on the comparisons or conditions you need to satisfy.

2. The Slow and Fast Pointers

In this method, two pointers move at different speeds. A common use is to detect cycles in a linked list. The fast pointer moves two steps at a time while the slow pointer moves one step at a time.

Step-by-Step Example

Let’s illustrate the Left and Right pointer method with a classic problem: Finding two numbers in a sorted array that add up to a given target.

Problem Statement

Given a sorted array nums = [1, 2, 3, 4, 6] and a target sum of 6, find the indices of the two numbers that add up to the target.

Step 1: Initialize Pointers

Start by setting the left pointer to the start of the array and the right pointer to the end:

left = 0; 
right = 4; // index of the last element

Step 2: Loop Until the Pointers Meet

While left < right, calculate the sum of the numbers at these indices:

sum = nums[left] + nums[right];

Step 3: Compare the Sum to Target

If the sum equals the target, you've found a pair. If the sum is less than the target, increment the left pointer to increase the sum:

if (sum == target) {
    return [left, right];
} else if (sum < target) {
    left++;
} else {
    right--;
}

Step 4: Conclusion

This process continues until the left pointer is no longer less than the right pointer. If a pair is found, return their indices; otherwise, return that no such pair exists.
