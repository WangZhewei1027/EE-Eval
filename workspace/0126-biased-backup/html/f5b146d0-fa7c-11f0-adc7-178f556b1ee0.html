Bellman-Ford Algorithm

Bellman-Ford is a dynamic programming algorithm used to find the shortest paths from a source node to all other nodes in a weighted graph. It can handle negative weight edges and negative cycles.

The algorithm works by iteratively relaxing all edges in the graph, and then repeatedly applying the algorithm to the updated graph.

Here is the step-by-step process:

  1. Initialize the distance to the source node as 0, and all other nodes as infinity.
  2. Relax all edges in the graph. This involves updating the distance to each node based on the shortest path to its neighbors.
  3. Repeat step 2 until all nodes have been processed.
  4. Finally, the algorithm returns the shortest distances from the source node to all other nodes.
