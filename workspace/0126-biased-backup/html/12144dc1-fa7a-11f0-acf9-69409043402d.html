Breadth-First Search (BFS) Interactive Demo

Instructions:
  • Define a directed or undirected graph by entering edges below.
  • Edges must be entered one per line in the format: NodeA NodeB, e.g. A B.
  • Nodes are case-sensitive and can be alphanumeric strings without spaces.
  • Select whether the graph is directed or undirected.
  • Enter the start node for BFS traversal.
  • Click Load Graph to parse and visualize the graph as adjacency list.
  • Use the step controls to run BFS step-by-step or automatically.
  • You can clear and redefine the graph anytime.
