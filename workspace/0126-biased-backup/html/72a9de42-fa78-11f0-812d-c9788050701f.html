About Radix Sort

Radix Sort is a non-comparative sorting algorithm that processes integer numbers by grouping them by individual digits which share the same significant position and value. It sorts numbers digit by digit starting from the least significant digit (rightmost) to the most significant digit (leftmost).

The algorithm uses counting sort as a subroutine to sort the digits at each position. This visualization shows how numbers are distributed into buckets based on each digit and then collected back into an array in sorted order.
