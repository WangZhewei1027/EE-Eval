Bucket Sort is a distribution sorting technique that divides elements into several groups called buckets. Each bucket is then sorted individually, either using another sorting algorithm or recursively applying bucket sort.

Here, numbers are placed into their respective buckets based on value ranges. After distributing, the elements inside each bucket get sorted and are finally concatenated in order, producing a fully sorted sequence.

Observe how the items glide fluidly between the original array and their buckets — an elegant dance orchestrated by logic and order.
