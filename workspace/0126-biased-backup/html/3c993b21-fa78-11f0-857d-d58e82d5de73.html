A mutex (mutual exclusion) is a synchronization primitive used to avoid simultaneous access to a critical section by multiple threads. Only one thread can lock the mutex, gaining exclusive access, while others must wait. This visualization shows threads as pendulums circling around the critical section, which is locked or unlocked by toggling the mutex state, demonstrating safe coordination in concurrent programming.
