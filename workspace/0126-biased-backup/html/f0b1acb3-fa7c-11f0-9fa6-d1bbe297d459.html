Heap (Max) - Comprehensive Guide

Introduction to Heaps

A heap is a specialized tree-based data structure that satisfies the heap property. In a max heap, for any given node, the value of that node is greater than or equal to the values of its children (if they exist). This makes the root node always contain the maximum value in the heap.

Heaps are commonly implemented as binary heaps, where each node has at most two children. Binary heaps are complete binary trees, meaning all levels are fully filled except possibly the last level, which is filled from left to right.

Key Property: In a max heap, the value of each node is greater than or equal to the values of its children.

Heap Representation

Heaps are typically represented using arrays, which provides an efficient storage mechanism and allows for easy navigation between parent and children:

  • For a node at index i (0-based indexing):
    • Parent index = Math.floor((i - 1) / 2)
    • Left child index = 2i + 1
    • Right child index = 2i + 2
  • The root of the tree is at index 0
  • The last leaf node is at index n-1 (where n is the size of the heap)

Example: Array Representation

Consider the following max heap:

        100
       /   \
     19     36
    / \    /
   17 3  25

This would be represented as the array: [100, 19, 36, 17, 3, 25]

Heap Operations

1. Insertion (Heapify Up)

To insert a new element into a max heap:

  1. Add the new element at the end of the array (as the last leaf)
  2. Compare the added element with its parent
  3. If the element is greater than its parent, swap them
  4. Continue this process up the tree until the heap property is satisfied

Insertion Example

Inserting 50 into our previous heap:

  1. Add 50 at the end: [100, 19, 36, 17, 3, 25, 50]
  2. Compare 50 (index 36) with its parent 36 (index 2)
  3. 50 > 36, so swap them: [100, 19, 50, 17, 3, 25, 36]
  4. Compare 50 (now at index 2) with its parent 100 (index 0)
  5. 50 ≤ 100, so we stop

2. Extract Max (Heapify Down)

To remove and return the maximum element (root):

  1. Store the root value to return later
  2. Replace the root with the last element in the array
  3. Remove the last element from the array
  4. Compare the new root with its children
  5. If it's smaller than either child, swap it with the larger child
  6. Continue this process down the tree until the heap property is satisfied

Extract Max Example

Extracting from our previous heap [100, 19, 50, 17, 3, 25, 36]:

  1. Store 100 (to return)
  2. Replace root with last element 36: [36, 19, 50, 17, 3, 25]
  3. Compare 36 with its children 19 and 50
  4. 50 is larger than 36, so swap them: [50, 19, 36, 17, 3, 25]
  5. Compare 36 (now at index 2) with its child 25
  6. 36 > 25, so we stop

Heap Construction

There are two main approaches to build a max heap from an unsorted array:

1. Successive Insertions (O(n log n))

This approach builds the heap by inserting elements one by one:

function buildHeapInsertion(arr) { let heap = []; for (let num of arr) { heap.push(num); // Heapify up from the last element let i = heap.length - 1; while (i > 0 && heap[Math.floor((i-1)/2)] < heap[i]) { [heap[Math.floor((i-1)/2)], heap[i]] = [heap[i], heap[Math.floor((i-1)/2)]]; i = Math.floor((i-1)/2); } } return heap; }

2. Heapify Down from Bottom (O(n))

This more efficient approach starts from the first non-leaf node and heapifies down:

function buildHeapFloyd(arr) { // Start from the first non-leaf node for (let i = Math.floor(arr.length/2) - 1; i >= 0; i--) { heapifyDown(arr, arr.length, i); } return arr; } function heapifyDown(arr, n, i) { let largest = i; let left = 2*i + 1; let right = 2*i + 2; if (left < n && arr[left] > arr[largest]) largest = left; if (right < n && arr[right] > arr[largest]) largest = right; if (largest !== i) { [arr[i], arr[largest]] = [arr[largest], arr[i]]; heapifyDown(arr, n, largest); } }

Applications of Max Heaps

  • Priority Queues: Max heaps efficiently support extract-max and insert operations
  • Heap Sort: A comparison-based sorting algorithm that uses a heap
  • Graph Algorithms: Used in algorithms like Dijkstra's and Prim's for efficient minimum/maximum extraction
  • Selection Algorithms: Finding the kth largest/smallest element efficiently

Heap Sort Algorithm

Heap sort uses the properties of a heap to sort an array:

  1. Build a max heap from the input array
  2. Swap the root (maximum value) with the last element
  3. Reduce the heap size by 1 (excluding the last element)
  4. Heapify the root of the tree
  5. Repeat until the heap is empty
function heapSort(arr) { // Build max heap for (let i = Math.floor(arr.length/2) - 1; i >= 0; i--) { heapifyDown(arr, arr.length, i); } // Extract elements one by one for (let i = arr.length - 1; i > 0; i--) { // Move current root to end [arr[0], arr[i]] = [arr[i], arr[0]]; // Heapify the reduced heap heapifyDown(arr, i, 0); } return arr; }

Time Complexity Analysis

Operation Time Complexity
Insert O(log n)
Extract Max O(log n)
Get Max O(1)
Build Heap (Floyd's method) O(n)
Heap Sort O(n log n)
