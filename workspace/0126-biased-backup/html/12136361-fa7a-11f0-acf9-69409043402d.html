Implementation Selection

Select the underlying implementation of the priority queue:

Insert Elements

Insert one or multiple elements into the queue. Format: value,priority
Example: task1,7 or multiple lines with one per element.



Priority: higher numeric value = higher priority

Remove Elements


Extract Max: removes and returns the element with highest priority
Peek Max: returns without removing
Clear All: empties the queue

Queue Contents & Operations




Changing priority or removing by value operates on the first matching value.
Increment/Decrement adjust all priorities.
Internal Data Structure print shows implementation-specific details.

Bulk & Advanced Operations




Bulk insert and benchmark facilitate testing performance.
Search finds all items with matching value.
Export/import queue state for saving/loading.
