Linear Regression

Understanding the best-fit line

Linear regression finds the straight line that best explains the relationship between two variables. Here, points represent observations — the line minimizes the overall vertical error (least squares).

Least squares • Slope & Intercept

The line is estimated analytically using the formula:

slope = cov(x,y) / var(x) • intercept = mean(y) − slope × mean(x)
