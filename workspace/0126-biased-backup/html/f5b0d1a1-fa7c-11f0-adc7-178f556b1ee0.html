Bucket Sort

Bucket sort is a comparison-based sorting algorithm that works by dividing the input data into several 'buckets' or bins, and then sorting each bucket individually.

Bucket sort is a relatively simple algorithm with a time complexity of O(n + k), where n is the number of elements in the input data and k is the number of buckets.

Theory

Bucket sort works by dividing the input data into k buckets, where each bucket is assigned a unique integer from 1 to k. The input data is then sorted individually by iterating through each bucket and placing each element in its correct position.

Here's a step-by-step textual description of the algorithm:

  1. Divide the input data into k buckets, where each bucket is assigned a unique integer from 1 to k.
  2. Sort each bucket individually using a comparison-based sorting algorithm (such as selection sort or insertion sort).
  3. Combine the sorted buckets to produce the final sorted output.

Algorithm Explanation

Bucket sort is implemented using the following steps:

  1. Initialize an empty array to store the buckets.
  2. For each element in the input data, create a bucket and assign it a unique integer from 1 to k.
  3. Sort the bucket using a comparison-based sorting algorithm.
  4. Combine the sorted bucket with the previous bucket to produce the final sorted output.

Example

Here's an example of bucket sort in action:

    // Input data: [5, 2, 8, 3, 1, 6, 4]
    // Bucket sort parameters: k = 3

    // Create k buckets
    let buckets = [];
    for (let i = 0; i < k; i++) {
        buckets.push([]);
    }

    // Divide input data into k buckets
    for (let i = 0; i < n; i++) {
        buckets[Math.floor(i / k)].push(i % k);
    }

    // Sort each bucket individually
    for (let i = 0; i < k; i++) {
        buckets[i].sort((a, b) => a - b);
    }

    // Combine sorted buckets to produce final sorted output
    let sortedOutput = [];
    for (let i = 0; i < k; i++) {
        sortedOutput.push(...buckets[i]);
    }

    console.log(sortedOutput);
