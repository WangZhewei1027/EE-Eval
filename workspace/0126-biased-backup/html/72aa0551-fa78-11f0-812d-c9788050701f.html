Linear Search

A simple yet fundamental search algorithm that checks each element in sequence until it finds the target value.

About Linear Search

Linear search is the simplest search algorithm. It sequentially checks each element of the list until a match is found or the whole list has been searched.

Time Complexity: O(n) in the worst case, as it may need to scan the entire array.

Space Complexity: O(1), as it requires no additional space beyond the input array.

While not the most efficient for large datasets, linear search is easy to implement and works well for small or unsorted collections.
