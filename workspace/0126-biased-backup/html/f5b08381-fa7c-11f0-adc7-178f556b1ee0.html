Bubble Sort

Bubble sort is a simple sorting algorithm that repeatedly steps through the list, compares adjacent elements, and swaps them if they are in the wrong order.

Here's a step-by-step explanation of the algorithm:

  1. Start with the first two elements in the list.
  2. Compare the first element with the second element and swap them if they are in the wrong order.
  3. Repeat the process for each pair of adjacent elements in the list.
  4. Continue this process until the entire list has been sorted.
