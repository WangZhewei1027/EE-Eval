Big-Omega Notation

Big-Omega Notation is a way of measuring the time or space complexity of an algorithm.

Time Complexity

Time Complexity is the amount of time an algorithm takes to complete, usually expressed as O(n) or O(log n).

Example 1: Linear Search

Linear Search is a simple search algorithm that checks each element in the array one by one.

                        function linearSearch(arr, target) {
                            for (let i = 0; i < arr.length; i++) {
                                if (arr[i] === target) {
                                    return i;
                                }
                            }
                            return -1;
                        }

Space Complexity

Space Complexity is the amount of memory an algorithm uses, usually expressed as O(n) or O(log n).

Example 2: Hash Table

Hash Table is a data structure that stores key-value pairs using a hash function.

                        class HashTable {
                            function create() {
                                this.size = 10;
                                this.table = new Array(this.size);
                            }

                            function insert(key, value) {
                                let index = this.hash(key);
                                if (this.table[index] === undefined) {
                                    this.table[index] = [];
                                }
                                this.table[index].push([key, value]);
                            }

                            function search(key) {
                                let index = this.hash(key);
                                if (this.table[index] === undefined) {
                                    return null;
                                }
                                for (let i = 0; i < this.table[index].length; i++) {
                                    if (this.table[index][i][0] === key) {
                                        return this.table[index][i][1];
                                    }
                                }
                                return null;
                            }

                            function hash(key) {
                                let hash = 0;
                                for (let i = 0; i < key.length; i++) {
                                    hash += key.charCodeAt(i);
                                }
                                return hash;
                            }
                        }
