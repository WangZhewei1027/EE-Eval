OSI Model

Application
This layer is responsible for network services to applications.
Presentation
This layer translates data between a networking service and an application.
Session
This layer manages sessions between applications.
Transport
This layer provides end-to-end communication services for applications.
Network
This layer handles the routing of data.
Data Link
This layer provides node-to-node data transfer and error correction.
Physical
This layer transmits raw bit streams over a physical medium.
