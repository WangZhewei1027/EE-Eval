Linked List

Linked List is a data structure in which elements are linked together through pointers.

A linked list is a linear collection of elements where each element points to the next element.

Key characteristics of a linked list include:

  • Each element is called a node or element.
  • A node contains two parts:
    • A data field (value),
    • A pointer to the next node.
  • Nodes are linked together through pointers, allowing for efficient insertion and deletion of elements.

Linked List Theory

Linked lists are dynamic data structures, meaning they can grow or shrink dynamically as elements are added or removed.

They are suitable for applications where the number of elements is not fixed and can change frequently.

Linked lists are also suitable for applications where random access is required, such as databases and file systems.

Linked List Algorithms

There are several algorithms for manipulating linked lists, including:

  • Insertion and deletion of elements at the beginning and end of the list.
  • Insertion and deletion of elements in the middle of the list.
  • Traversal of the list (e.g. in-order, pre-order, post-order).

Linked List Implementation

A linked list can be implemented using a series of nodes, where each node contains a data field and a pointer to the next node.

The nodes are linked together through pointers, allowing for efficient insertion and deletion of elements.

Here is a simple implementation of a linked list in JavaScript:

    class Node {
        constructor(data) {
            this.data = data;
            this.next = null;
        }
    }
    
    class LinkedList {
        constructor() {
            this.head = null;
        }
        
        add(data) {
            let newNode = new Node(data);
            if (this.head === null) {
                this.head = newNode;
            } else {
                let current = this.head;
                while (current.next !== null) {
                    current = current.next;
                }
                current.next = newNode;
            }
        }
        
        print() {
            let current = this.head;
            while (current !== null) {
                console.log(current.data);
                current = current.next;
            }
        }
    }
    
    let list = new LinkedList();
    list.add('A');
    list.add('B');
    list.add('C');
    list.print();
