Binary Tree

Binary Tree is a data structure in which each node has at most two children, referred to as the left child and the right child. This tree is named after its inventor, George Boole, who used a binary tree to model Boolean logic.

Binary trees are used extensively in computer science for storing and retrieving data. They are particularly useful for organizing and searching large amounts of data.

Types of Binary Trees

  • Binomial Tree
  • AVL Tree
  • BT (Binary Search Tree)

Types of Nodes

  • Root Node
  • Left Child
  • Right Child
  • Leaf Node

Insertion and Deletion

Insertion: When a new node is inserted into a binary tree, it is inserted at the root node. The left child and right child are then updated accordingly.

Deletion: When a node is deleted from a binary tree, it is deleted from the root node. The left child and right child are then updated accordingly.

Traversal

There are three types of traversal in binary trees: Inorder Traversal, Preorder Traversal, and Postorder Traversal.

  • Inorder Traversal: The traversal is done in the order of left, root, and right.
  • Preorder Traversal: The traversal is done by visiting the root, then the left child, and finally the right child.
  • Postorder Traversal: The traversal is done by visiting the left child, then the root, and finally the right child.

Balancing

Balancing is the process of maintaining the balance of a binary tree. This is necessary to ensure that the tree remains approximately balanced, which can improve the performance of certain algorithms.

AVL Trees

AVL Trees are a type of binary search tree that ensures the balance of the tree. They are particularly useful for storing and retrieving data that requires efficient searching and inserting.

BT (Binary Search Tree)

Here is an example of a binary tree in Python:

    class Node:
        def __init__(self, value):
            self.value = value
            self.left = None
            self.right = None

    def create_binary_tree(self, values):
        if not values:
            return None

        root = Node(values[0])
        current = root

        for value in values[1:]:
            current.left = Node(value)
            current = current.left

        current.right = Node(value)
        current = current.right

        return root

    def inorder_traversal(self, root):
        if root:
            self.inorder_traversal(root.left)
            print(root.value, end=" ")
            self.inorder_traversal(root.right)

    def preorder_traversal(self, root):
        if root:
            print(root.value, end=" ")
            self.preorder_traversal(root.left)
            self.preorder_traversal(root.right)

    def postorder_traversal(self, root):
        if root:
            self.postorder_traversal(root.left)
            self.postorder_traversal(root.right)
            print(root.value, end=" ")

    def balance_tree(self, root):
        if not root:
            return root

        left_height = self.get_height(root.left)
        right_height = self.get_height(root.right)

        if abs(left_height - right_height) > 1:
            if left_height > right_height:
                root.left = self.balance_tree(root.left)
            else:
                root.right = self.balance_tree(root.right)

        return root

    def get_height(self, root):
        if not root:
            return 0
        else:
            return 1 + max(self.get_height(root.left), self.get_height(root.right))

    def print_tree(self, root):
        if root:
            self.print_tree(root.left)
            print(root.value, end=" ")
            self.print_tree(root.right)

    def create_binary_tree_values(self, values):
        if not values:
            return None

        root = Node(values[0])
        current = root

        for value in values[1:]:
            current.left = Node(value)
            current = current.left

        current.right = Node(value)
        current = current.right

        return root

    def insert(self, root, value):
        if not root:
            return Node(value)
        else:
            if value < root.value:
                if not root.left:
                    root.left = Node(value)
                else:
                    self.insert(root.left, value)
            else:
                if not root.right:
                    root.right = Node(value)
                else:
                    self.insert(root.right, value)

    def delete(self, root, value):
        if not root:
            return root
        elif value < root.value:
            root.left = self.delete(root.left, value)
        elif value > root.value:
            root.right = self.delete(root.right, value)
        else:
            if not root.left and not root.right:
                return None
            elif not root.left:
                return root.right
            elif not root.right:
                return root.left
            else:
                min_val = self.find_min(root.right)
                root.value = min_val
                root.right = self.delete(root.right, min_val)
        return root

    def find_min(self, root):
        current = root
        while current.left:
            current = current.left
        return current.value

    def print_tree_values(self, root):
        if root:
            self.print_tree_values(root.left)
            print(root.value, end=" ")
            self.print_tree_values(root.right)
