OPEN SYSTEMS INTERCONNECTION
MODEL

A conceptual framework used to understand and implement standardized communication protocols in seven distinct layers.

Application
End-user software interacting with the network
Presentation
Data translation, encryption, and compression
Session
Controls sessions and connections between devices
Transport
End-to-end data transfer and reliability
Network
Routing, addressing and packet forwarding
Data Link
Error detection, framing, and physical addressing
Physical
Transmission of raw bitstream over physical medium
