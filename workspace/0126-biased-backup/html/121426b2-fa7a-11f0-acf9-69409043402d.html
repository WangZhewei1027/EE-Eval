Step 5: Notes and Explanation

This interactive tool visualizes the classic ternary search algorithm exploring a sorted integer array to find a target value (Exact mode), or the ternary search approach used to find the maximum of a unimodal discrete function (Max mode).

Exact mode: The array must be sorted. The search repeatedly divides the range into three parts and compares the values at two midpoints to decide which range to keep searching in.

Max mode: The array represents a unimodal function's sampled values. The search tries to find the maximum element's index efficiently using ternary search by comparing function values at two midpoints.

You may load your own array and target value, step forward and backward through the search steps, auto-play with adjustable speed, jump to any step, and manually override the search range to experiment fully.
