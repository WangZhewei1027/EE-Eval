Support Vector Machine

An aesthetic illustration depicting the classic SVM classifier, showcasing data points separated by the optimal hyperplane and the margin-defining support vectors.

Positive Class (+1)
Negative Class (-1) Support Vectors Hyperplane

The Support Vector Machine algorithm separates two classes of data points with a hyperplane that maximizes the margin – the distance between the closest points of each class, called support vectors. Here, green and red points represent two classes, and the glowing circles highlight those critical support vectors that define the boundary. The center line is the optimal hyperplane that splits the classes with the maximum margin.
