Interactive Exploration of Type Systems

This tool lets you experiment with different type system features, create custom types, build expressions, and see how typing rules apply step by step.

1. Select Type System Features











  
2. Define Custom Types

  
Define new types as algebraic data types using the syntax below, if enabled:

  
type TypeName a b = Constructor1 Type1 Type2 | Constructor2 Type3 | ...

  
Example: type Maybe a = Just a | Nothing

  
  

  
  
  










  
3. Create Expressions

  
Create expressions using variables, literals, functions, and type constructors.

  
Supported expression syntax (depends on enabled features):

  
- Literals: 1, true, false, "text"
- Variables: x, y, z ...
- Lambda: \x -> expr
- Application: f x
- Let: let x = expr1 in expr2
- Constructors: Just 5, Nothing
- Polymorphic functions: id, const (pre-defined)
