Exponential Search Interactive Demo

This tool lets you explore, build, and run the Exponential Search algorithm step-by-step on a sorted array. You can input your own array, adjust the value to search, control the search speed, step through each phase, and experiment with different binary search variants used in the algorithm.

Interactive Workflow:

  • Load a valid sorted array.
  • Set the target value and select binary search variant.
  • Click Start Search to begin the Exponential Search step-through.
  • Use Step Forward and Step Backward to explore each detailed step or Auto-Play to run automatically.
  • Modify parameters mid-search and Reset All to restart fresh.
