Red-Black Tree

Visual Representation of Red-Black Tree Data Structure

Node Properties

The following properties are used to construct a Red-Black Tree:

  • Color (Red or Black): Indicates the color of the node.
  • Left Child: The node to the left of the current node.
  • Right Child: The node to the right of the current node.
  • Parent: The parent node of the current node.
  • Height: The height of the tree.

Insertion Order

Insertion order is the order in which nodes are inserted into the tree. The tree is always sorted in ascending order.

Traversal Order

Traversal order is the order in which nodes are visited during a traversal of the tree. The tree is always traversed in ascending order.

Red-Black Tree Properties

The following properties are used to construct a Red-Black Tree:

  • Each node has a color (Red or Black).
  • The root node is always Black.
  • The left and right children of a node are either Black or Red.
