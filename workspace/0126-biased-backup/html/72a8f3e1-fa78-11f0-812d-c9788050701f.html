What is a Multiset?

A multiset (also called a bag) is a generalization of a set that allows for multiple instances of its elements. Unlike classical sets where each element appears exactly once, elements in a multiset can appear any number of times.

The multiplicity of an element is the number of times it appears in the multiset. For example, in our visualization above, element "A" has multiplicity 2, "B" has multiplicity 1, and "C" has multiplicity 3.

Multisets are used in many areas of mathematics and computer science, including combinatorics, database theory, and graph theory.

🔢

Cardinality

The cardinality of a multiset is the sum of the multiplicities of all its elements, representing the total number of element occurrences.

➕

Operations

Multisets support operations like union, intersection, and sum, each handling multiplicities in specific ways while combining multisets.

🔄

Applications

Used in combinatorics, database systems (as bags), network packet counting, and anywhere multiple instances of elements matter.
