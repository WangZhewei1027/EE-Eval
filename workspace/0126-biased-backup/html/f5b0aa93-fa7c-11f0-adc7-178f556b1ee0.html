Counting Sort

Counting Sort is a sorting algorithm that uses the range of input values to determine the number of distinct elements. It works by counting the number of elements that are less than or equal to a given value, and then using this count to determine the range of input values.
The Counting Sort algorithm works as follows: 1. Sort the input array in ascending order. 2. Create an array of counts, where each element represents the number of elements that are less than or equal to the corresponding element in the input array. 3. Iterate through the input array and increment the corresponding count in the counts array. 4. Use the counts array to determine the range of input values. 5. Sort the input array based on the values in the counts array.
Counting Sort is a simple and efficient algorithm for sorting large datasets. However, it has a time complexity of O(n + k), where n is the number of elements in the input array and k is the range of input values.

Example

Suppose we have an array of integers from 3 to 9. If we sort this array in ascending order, we get: [3, 4, 5, 6, 7, 8, 9].

If we want to sort this array based on the values in the counts array, we get: [3, 4, 5, 6, 7, 8, 9].
