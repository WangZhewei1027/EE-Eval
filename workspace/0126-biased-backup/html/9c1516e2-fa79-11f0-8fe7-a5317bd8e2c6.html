P vs NP Interactive Playground

This single-file interactive demo explores P vs NP concepts with many interactive controls. The UI is intentionally plain: black text on white background, basic controls. Use the controls to build instances, choose solvers, step through algorithms, run experiments, and explore reductions and verification.
