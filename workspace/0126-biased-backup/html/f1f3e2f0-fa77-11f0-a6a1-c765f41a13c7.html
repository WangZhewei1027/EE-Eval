Ordered • Contiguous • Indexed

An array is a linear, fixed-size collection of elements stored contiguously in memory. Each element is accessible by its index — a number representing its offset from the start. Below is a stylized representation that highlights sequential access and indexed addressing.

Conceptual Notes

Elements are stored next to each other. Indexing calculates an element's location: address_of_first + index * size_of_element. Traversal visits each element, and random access directly targets an index.
