Breadth-First Search (BFS)

Breadth-First Search (BFS) is a graph traversal algorithm that visits all nodes at a given depth before moving on to the next depth level.

BFS is a popular algorithm in computer science for finding shortest paths between nodes in an unweighted graph.

BFS works by exploring all nodes at the current depth level before moving on to the next depth level.

Here is a step-by-step textual description of the BFS algorithm:

  1. Start at the source node.
  2. Explore all neighbors of the source node.
  3. Add each neighbor to the result set.
  4. Repeat steps 2-4 until all nodes at the current depth level have been visited.
  5. Repeat steps 1-4 until all nodes have been visited.

Here is a simple example of BFS in Python:

```python from collections import deque def bfs(graph, start): result = [] queue = deque([start]) while queue: node = queue.popleft() result.append(node) for neighbor in graph[node]: if neighbor not in queue: queue.append(neighbor) return result # Example graph graph = { 'A': ['B', 'C'], 'B': ['A', 'D', 'E'], 'C': ['A', 'F'], 'D': ['B'], 'E': ['B', 'F'], 'F': ['C', 'E'] } start_node = 'A' result = bfs(graph, start_node) print("Shortest path:", result) ```
