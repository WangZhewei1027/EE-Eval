How Counting Sort Works

Counting sort is an algorithm for sorting a collection of objects according to keys that are small integers. It operates by:

  1. Counting the number of objects that have each distinct key value
  2. Using arithmetic on those counts to determine the positions of each key value in the output sequence
  3. Copying each object to its correct position in the output array

The algorithm runs in O(n + k) time where n is the number of elements and k is the range of the input.
