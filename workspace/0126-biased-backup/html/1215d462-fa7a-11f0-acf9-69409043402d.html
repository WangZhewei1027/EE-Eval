NoSQL Interactive Demo

This demo simulates an in-memory document-oriented NoSQL database with powerful interactive controls. You can create collections, insert, update, delete, and query documents, define indexes, perform complex queries, and explore NoSQL concepts.

Database Setup
Document Operations
Query Interface

Enter a JSON query object. Supports simple comparison with operators $eq, $ne, $gt, $lt, $gte, $lte, $in, $nin and logical operators $and, $or, $not.

Indexes

Create indexes on fields to accelerate query simulation. (This demo doesn't execute real index acceleration but showcases management and effect on state.)

Aggregation Operations

Perform simple aggregation pipeline on selected collection. Supports $match, $group, $sort, $limit.
