Types that depend on values, allowing for more expressive and precise type information, often used in theorem proving and advanced type systems.

A Type System is a set of rules that assigns a property called a type to the various constructs in a computer program, such as variables, expressions, functions, or modules. Types classify data and help prevent errors by constraining operations to valid combinations. From primitive to dependent types, the system provides a framework for both safety and expressiveness in programming languages.
