Batch Operations and Persistence


Supported batch commands:
- pushFront VALUE
- pushBack VALUE
- popFront
- popBack
- peekFront
- peekBack
- clear
- removeValue VALUE
- countValue VALUE
- findValue VALUE
- insertAt INDEX VALUE
- removeAt INDEX
Lines starting with # are ignored.
JSON files store the deque array and max size.
