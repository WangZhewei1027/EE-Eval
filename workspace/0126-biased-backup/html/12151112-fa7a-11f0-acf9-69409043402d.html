Big-Theta Notation Interactive Explorer

1. Definition and Overview

Θ(g(n)) = { f(n) | there exist positive constants c1, c2, n0 such that for all n ≥ n0, c1 × g(n) ≤ f(n) ≤ c2 × g(n) }
Big-Theta notation describes a tight bound on the growth rate of functions.

2. Build & Compare Functions

Create functions f(n) and g(n) using predefined terms and test if f(n) = Θ(g(n)).





3. Step-by-Step Exploration

Explore the constants c1, c2 and n0 used to establish Θ-bound.
Adjust constants and see bounds and inequalities for chosen n values.

4. Evaluate Functions at Specific n

Test computed values of f(n) and g(n) at chosen n.

Help / Instructions

How to use this tool:
1. Select terms and enter coefficients for f(n) and g(n). The order of coefficients corresponds to the order of terms shown.
2. Specify the range of n values over which to test f(n) ≤ c2·g(n) and f(n) ≥ c1·g(n).
3. Click "Evaluate f(n) versus Θ(g(n))" to determine whether f(n) = Θ(g(n)) seems plausible within tested range.
4. Then explore constants c1, c2, and n0 explicitly to see inequalities detailed.
5. Use Evaluate Functions at Specific n to check numeric values.

Notes:
- Multiple terms are summed to create the functions.
- If g(n) becomes zero or negative in range, results may be invalid.
- The tool uses sampled n values and simple numerical checks.
