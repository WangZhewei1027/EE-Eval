How it works — in one glance

Symmetric cryptography uses the same secret key to both encrypt and decrypt messages. In this simplified visualization, you can see a plaintext word travel through a shared key producing ciphertext — and the same key allows the original message to be recovered.

Note: This demo is illustrative. Real symmetric systems use strong algorithms (AES, ChaCha20, etc.) and careful key management. Still, the central idea remains: one key — two directions.
