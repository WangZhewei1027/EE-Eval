Application Layer

The top layer where network applications and protocols like HTTP, FTP, and SMTP operate. This layer provides services directly to user applications.

Transport Layer

Responsible for end-to-end communication and error recovery. TCP provides reliable, ordered delivery, while UDP offers faster but unreliable transmission.

Internet Layer

The IP (Internet Protocol) layer handles addressing and routing of packets across network boundaries. It's connectionless and doesn't guarantee delivery.

Network Access

The physical network components that handle transmission of raw bits over the network medium (cables, WiFi). Includes MAC addresses and physical signaling.
