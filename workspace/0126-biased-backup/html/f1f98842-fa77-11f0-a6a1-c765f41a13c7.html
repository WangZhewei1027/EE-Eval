How Backpropagation Works
Illustrative steps

Backpropagation computes gradients of the loss with respect to model parameters by applying the chain rule backward through the network. Watch the signal (cyan) flow forward, see the loss form, and observe gradients (magenta) propagate backward to update weights.

1
Forward Pass
Inputs activate hidden units; signals reach the output to produce prediction ŷ.
2
Loss Computation
A loss quantifies the error between prediction and target — this drives learning.
3
Backward Pass
Gradients flow back through the network via the chain rule, computing ∂L/∂w for each weight.
4
Weight Update
Weights are adjusted (w ← w − η ∂L/∂w), nudging the model toward lower loss.
Quick Insight
Chain rule in action

If y = f(g(h(x))) then dy/dx = f'(g(h(x))) · g'(h(x)) · h'(x). Backpropagation applies this repeatedly across layers to compute gradients efficiently.
