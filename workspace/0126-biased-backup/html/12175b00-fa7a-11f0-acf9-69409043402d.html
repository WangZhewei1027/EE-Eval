Decision Tree Explorer

This tool lets you build, edit, and explore decision trees interactively. You can create nodes representing questions or decisions, connect nodes via branches with conditions, and simulate decision flow.

1. Create / Edit Nodes

Nodes represent points in your decision tree. Each node has a label (question or decision) and can have multiple branches connecting to other nodes.

2. Add / Manage Branches (Edges)

Define edges (branches) from one node to another with a condition label.

Note: Branch conditions help guide decisions from one node to another.

3. Simulation Mode

Explore the decision tree by simulating choices from a start node. You can select a start node and navigate through decisions.
