How Quick Sort Works

Quick Sort is a divide-and-conquer algorithm that works by selecting a 'pivot' element from the array and partitioning the other elements into two sub-arrays according to whether they are less than or greater than the pivot.

The sub-arrays are then sorted recursively. This continues until the base case of a sub-array with 0 or 1 elements is reached, which by definition is already sorted.

The key process is the partitioning step, which rearranges the array in-place so that all elements less than the pivot come before it, and all elements greater come after.
