About Red-Black Trees

A Red-Black Tree is a kind of self-balancing binary search tree where each node contains an extra bit representing "color" (either red or black). This color bit ensures the tree remains approximately balanced during insertions and deletions.

Properties:

  • Each node is either red or black.
  • The root is always black.
  • All leaves (NIL or null nodes) are black.
  • If a node is red, then both its children are black (no two reds in a row).
  • Every path from a given node to any of its descendant leaves contains the same number of black nodes.

Operations:

This demo supports insertions. After each insertion, the tree automatically rebalances and recolors to maintain Red-Black properties.
