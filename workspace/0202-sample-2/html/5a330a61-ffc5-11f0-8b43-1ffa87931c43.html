What is Heap Sort?

Heap Sort is a comparison-based sorting algorithm that uses a binary heap data structure. It's an efficient sorting algorithm with O(n log n) time complexity in the worst, average, and best cases.

The two main steps of heap sort are:

  1. Build a max heap from the input data.
  2. Repeatedly remove the largest element (root of the heap) and place it at the end of the array, then rebuild the heap with the remaining elements.

This visualization demonstrates how the values "bubble down" in the heap during the heapify process and how the sorted portion grows from the end of the array.
