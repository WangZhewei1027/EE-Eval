Binary Tree Interactive Demonstration



This demonstration shows a Binary Tree where each node has at most two children (left and right).

You can insert integer values into the tree. The tree will be displayed graphically, with each node shown as a circle containing its value.

Use the traversal buttons to see the order in which nodes are visited during different tree traversals:

  • Pre-order: Visit node, then left subtree, then right subtree.
  • In-order: Visit left subtree, then node, then right subtree.
  • Post-order: Visit left subtree, then right subtree, then node.
  • Level-order: Visit nodes level by level from top to bottom.
