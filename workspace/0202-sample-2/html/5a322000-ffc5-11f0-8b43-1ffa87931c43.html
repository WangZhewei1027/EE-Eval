About this demo:

A linked list is a linear data structure where each element (called a node) contains a value and a pointer/reference to the next node. This demo allows you to add, insert and remove nodes to see how the list changes.

  • Push: Add node to the end.
  • Unshift: Add node to the start.
  • Insert at Index: Insert node at specified position (0-based).
  • Pop: Remove node from the end.
  • Shift: Remove node from the start.
  • Remove at Index: Remove node at specified position.
