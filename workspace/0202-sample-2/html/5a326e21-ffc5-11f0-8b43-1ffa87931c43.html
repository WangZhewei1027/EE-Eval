What is a Hash Map?

A Hash Map is a data structure that stores key-value pairs allowing very fast access, insertion, and deletion of values based on keys. Internally, it uses a hash function to compute an index for placing data, enabling nearly constant time complexity on average.

How This Works (Simplified)

This demo uses a simple hash function that converts a string key into an index between 0 and 9. Each index stores a list of entries (to handle collisions using chaining). When adding or looking up keys, it:

  • Hashes the key into an index
  • At that index, searches if the key exists
  • Adds updates or retrieves the value as necessary
