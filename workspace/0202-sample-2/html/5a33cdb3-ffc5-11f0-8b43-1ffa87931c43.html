K-Nearest Neighbors (KNN) Interactive Demo

Class A points
Class B points
Unknown points (to classify)

Instructions:

  1. Click on canvas to add blue Class A points (left mouse button).
  2. Right-click to add orange Class B points.
  3. Shift + click to add "unknown" points to classify.
  4. The KNN algorithm will color unknown points based on their nearest neighbors.
  5. Adjust k to see classification change.
  6. Press "Clear" to reset.
