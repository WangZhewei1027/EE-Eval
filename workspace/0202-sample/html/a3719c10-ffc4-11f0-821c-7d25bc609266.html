Linear Regression: A Comprehensive Introduction

Linear Regression is one of the foundational algorithms in statistics and machine learning used to model the relationship between a dependent variable and one or more independent variables. It is widely used for predictive modeling and inferential analysis.

1. What is Linear Regression?

At its core, linear regression attempts to find a linear relationship between variables. Specifically, it models the dependent variable Y as a linear combination of one or more independent variables X plus some error (noise).

“Linear regression seeks to fit a straight line (or hyperplane, in multiple dimensions) through the data points that best predicts the target variable from the inputs.”

Simple Linear Regression (One Predictor)

When there is only one independent variable, the model is called simple linear regression. The relationship is modeled as:

Y = β₀ + β₁ X + ε

Where:
Y: Dependent (target) variable.
X: Independent (predictor) variable.
β₀: Intercept term (value of Y when X=0).
β₁: Slope coefficient (change in Y for a unit change in X).
ε: Error term accounting for noise or unmodeled factors.

Multiple Linear Regression

When there are multiple independent variables, the model generalizes to:

Y = β₀ + β₁X₁ + β₂X₂ + ... + βₚXₚ + ε

This fits a hyperplane in a p-dimensional predictor space.

2. The Goal of Linear Regression

The primary goal is to estimate the parameters β₀, β₁, ..., βₚ so the predicted values Ŷ are as close as possible to the actual Y values in the training data.

This is often framed as an optimization problem: find parameters that minimize the prediction error, typically using the sum of squared residuals (differences between observed and predicted values):

Minimize ∑i=1n (Yi − Ŷi)² = Minimize ∑i=1n (Yi − (β₀ + β₁Xi))²

This method is called Ordinary Least Squares (OLS).

3. The Mathematics Behind Linear Regression

Let's dive deeper into how the coefficients are computed in simple linear regression.

Formulation

Given a dataset of n observations {(X1, Y1), (X2, Y2), ..., (Xn, Yn)}, the goal is to estimate β₀ and β₁.

Calculating the Slope (β₁):

The formula for the slope is:

β₁ = ∑(Xi − 𝑋̄)(Yi − Ȳ)⁄∑(Xi − 𝑋̄)²

Where: 𝑋̄ and Ȳ are sample means of the X and Y variables respectively.

Calculating the Intercept (β₀):

Once β₁ is known, the intercept is computed as:

β₀ = Ȳ − β₁ 𝑋̄

Predictions

With estimates β₀ and β₁ we can predict new values:

Ŷ = β₀ + β₁ X

This line of best fit will minimize the sum of squared vertical distances between the observed points and the predicted line.

4. Why Use Linear Regression?

Linear regression is among the simplest supervised learning algorithms and the easiest to interpret. Benefits include:

  • Interpretability: Each coefficient shows the expected change in the target per unit increase in a feature.
  • Efficiency: Computationally fast to train and predict.
  • Foundation: Serves as a building block for many more complex algorithms.
  • Assumptions: When assumptions hold (linearity, homoscedasticity, independence, normality), estimates are unbiased and efficient.

5. Assumptions of Linear Regression

Linear regression relies on several underlying assumptions for the model and estimators to be valid:

1. Linearity
The relationship between the independent variables and dependent variable is linear.
2. Independence
The residuals (errors) are independent. No autocorrelation.
3. Homoscedasticity
Constant variance of the error terms across values of independent variables.
4. Normality of Errors
Errors are normally distributed (important for inference and hypothesis tests).
5. No multicollinearity (for multiple regression)
Independent variables are not highly correlated with each other.
Note: Violations of these assumptions can render the model inaccurate or invalid; diagnostic checks are typically performed after fitting.

6. Evaluating Model Performance

After fitting a linear regression model, performance is assessed using metrics such as:

Mean Squared Error (MSE)
The average of squared differences between observed and predicted values. Smaller values indicate better fit.
Root Mean Squared Error (RMSE)
The square root of MSE, interpretable in the original units of the target variable.
R-squared (Coefficient of Determination)
Measures the proportion of variance in the dependent variable explained by the model. Ranges from 0 to 1, with higher values indicating better explanatory power.

7. Step-by-Step Summary of Fitting a Simple Linear Regression Model

  1. Collect Data: Obtain paired observations (X, Y).
  2. Visualize: Plot data points to explore linearity.
  3. Calculate means: Compute 𝑋̄ and Ȳ.
  4. Calculate slope β₁: Use the formula for covariance over variance.
  5. Calculate intercept β₀: Use β₀ = Ȳ − β₁ 𝑋̄.
  6. Form the regression line: Ŷ = β₀ + β₁ X.
  7. Evaluate: Calculate residuals and metrics like MSE, R².
  8. Use: Make predictions for new data points.
