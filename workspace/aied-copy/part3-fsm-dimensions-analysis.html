📊 Table Interpretation

This table shows the mean and standard deviation (±) for each FSM dimension across all samples generated by each model. All scores are normalized using z-score transformation with 2.5× variance amplification to match the distribution characteristics of human evaluations (standard deviation ~10-20), mapped to a [15, 85] range.

Note: Zero-valued dimensions (Isomorphism, State Overlap, Transition Overlap) have been removed for clearer visualization.

Dimensions:

  • Node Count Similarity: How similar the number of states is between generated and ideal FSM
  • Edge Count Similarity: How similar the number of transitions is between FSMs
  • Structural (40%): Graph topology metrics (nodes, edges, connectivity)
  • Semantic (40%): Conceptual alignment and state naming similarity
  • Overall Score: Weighted combination of structural and semantic dimensions

🎯 Radar Chart Interpretation

Each radar chart visualizes a model's performance across five FSM dimensions. All scores are normalized using z-score transformation with 2.5× variance amplification to match human evaluation distributions, enabling fair visual comparison. A larger polygon area indicates better overall performance, while the shape reveals strengths and weaknesses:

  • Balanced polygon: Model performs consistently across all dimensions
  • Asymmetric polygon: Model excels in some dimensions but underperforms in others
  • Larger area: Higher overall quality and similarity to ideal FSMs

Compare the shapes across models to identify which models generate FSMs that are structurally sound and semantically meaningful. Zero-valued dimensions (Isomorphism, State/Transition Overlap) are excluded.
