Floyd–Warshall Algorithm — Interactive Demonstration

Floyd–Warshall computes shortest paths between all pairs of nodes in a weighted graph (works with directed graphs and handles negative edges if there are no negative cycles). Use the controls to edit the adjacency, step through the algorithm, or run it automatically. The "next" matrix lets you reconstruct shortest paths.
