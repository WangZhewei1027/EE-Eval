Prim's Algorithm

Prem's Algorithm is a number-solving algorithm that starts with the given number and works backwards until it reaches the beginning of the sequence. Here are the steps:

The first step is to initialize the current position of the sequence as the starting point, which is 0.

The second step is to generate all possible pairs of numbers from 0 to the current position.

The third step is to check if each pair has already been used before moving on to the next one. If it has, the algorithm returns -1. Otherwise, it moves on to the next pair and continues with the process until it reaches the beginning of the sequence.

The final step is to return the number at the end of the sequence.
