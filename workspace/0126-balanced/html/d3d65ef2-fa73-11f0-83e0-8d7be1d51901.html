Pseudocode (stable Counting Sort)
CountingSort(A):
  let min = min(A), max = max(A)
  let offset = min
  let range = max - min + 1
  create count[0..range-1] = {0}
  for i = 0 to A.length-1:
    count[A[i] - offset] += 1
  // cumulative counts
  for j = 1 to range-1:
    count[j] += count[j-1]
  // build output (stable): iterate A from right to left
  create output[A.length]
  for i = A.length-1 down to 0:
    idx = A[i] - offset
    position = count[idx] - 1
    output[position] = A[i]
    count[idx] -= 1
  return output
Notes
- Counting Sort runs in O(n + k) time where k is the range size (max-min+1).
- It is stable when we place elements into output from right to left using cumulative counts.
- For very large ranges relative to n, Counting Sort can be memory-inefficient.
- This demo supports negative values by applying an offset.
