About this demo

A doubly linked list stores nodes that know about both their previous and next neighbors. This allows forward and backward traversal in O(n) but with O(1) insertions/removals when you have a reference to the node.

  • Insert Head / Tail — add nodes at ends
  • Insert After Selected / Value — insert after a chosen node
  • Remove by Value / Index — delete nodes
  • Traverse → / Traverse ← — animate traversals
