A* Search Algorithm

How to use:

  1. Click "Set Start" then click a cell to set the starting point
  2. Click "Set End" then click a cell to set the destination
  3. Click "Set Wall" then click cells to add obstacles
  4. Click "Run A* Search" to find the shortest path

A* Search is a pathfinding algorithm that finds the shortest path between two points by combining the cost to reach a node (g) with a heuristic estimate of the cost to the goal (h). It prioritizes nodes with the lowest f = g + h value.
