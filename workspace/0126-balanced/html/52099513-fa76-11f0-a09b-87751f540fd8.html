Dynamic Programming Example

Problem: 1 + 2 + 3 + 4 + 5 = ?

Manual calculation: 1 + 2 + 3 + 4 + 5 = 15

Dynamic Programming Solution:

1. Initialize an array dp with 5 elements, all set to 0.

2. Iterate through the numbers 1 to 5.

3. For each number i, calculate dp[i] as the sum of dp[i - 1] and i.

4. Return dp[5] as the result.
