Amortized Analysis Demonstration: Dynamic Array (ArrayList) Resizing

What is Amortized Analysis?
Amortized analysis calculates the average time per operation over a worst-case sequence of operations. It is especially helpful when expensive operations happen infrequently but can otherwise distort average cost estimates.

Example: Dynamic Array (ArrayList) Expansion
Imagine a dynamic array that doubles its size when full. Most insert operations run in constant time O(1), but resizing and copying elements when the array is full is expensive O(n). Through amortized analysis, we show that the average cost per insertion is still O(1).
