Algorithm

Depth-First Search explores as far as possible along each branch before backtracking. This demo uses an explicit stack (iterative DFS). You can watch nodes get pushed to the stack, popped for exploration, and see the final path when the goal is reached (if reachable).
DFS(start, goal): stack = [start] parent[start] = null visited = set(start) while stack not empty: node = stack.pop() if node == goal: return reconstruct_path(parent,node) for neighbor in neighbors(node): if neighbor not in visited and not wall: visited.add(neighbor) parent[neighbor] = node stack.push(neighbor) end end end return "no path"

Controls & Info

- Wall: click/drag to paint walls. Erase: remove walls. Set Start/Goal: click a cell.
- Run: animates DFS automatically. Step: advances one DFS operation per press.
Tip: Use "Clear Visited" to keep walls/start/goal and run again.
