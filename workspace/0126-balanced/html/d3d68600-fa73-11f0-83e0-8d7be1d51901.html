How it works (LSD):
- For each digit from least significant to most, place numbers into buckets according to that digit.
- Then collect buckets in order to form the new array; repeat for next digit.
- Complexity: O(n * d) where d is number of digits (log_base(max)).

Algorithm (LSD Radix Sort)
// Pseudocode (LSD Radix Sort) radixSort(A, base): maxVal = max(A) d = number of digits of maxVal in base for digit = 0 to d-1: // Distribute into buckets 0..base-1 for each x in A: key = digitValue(x, digit, base) buckets[key].append(x) // Collect buckets in order (stable) A = concatenate buckets[0], buckets[1], ..., buckets[base-1] clear all buckets
Details
- This visualizer uses LSD (least significant digit) approach and supports different bases.
- All numbers are treated as non-negative. For negatives, you can offset values or handle signs separately.
