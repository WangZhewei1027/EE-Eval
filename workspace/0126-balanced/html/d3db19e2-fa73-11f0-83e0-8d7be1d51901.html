Dynamic typing means variables are not fixed to a single type; the type is determined at runtime and can change. Play with the variable below to see its type change and how operations behave.

  • In dynamically typed languages like JavaScript, x can hold any value and you can reassign it to a different type.
  • Some operators coerce types: + often concatenates strings, but - forces numeric conversion.
  • Equality == performs type coercion; === checks both value and type (strict).
  • You can add properties to objects at runtime; arrays and functions are also just values you can reassign.
