How it works
Ternary search repeatedly divides the search interval into three parts using two mid indices m1 and m2: m1 = l + ⌊(r-l)/3⌋ and m2 = r - ⌊(r-l)/3⌋. Compare the target with arr[m1] and arr[m2]: - If equal to either, we've found the target.
- If target < arr[m1], search left third: r = m1 - 1.
- If arr[m1] < target < arr[m2], search middle third: l = m1 + 1, r = m2 - 1.
- If target > arr[m2], search right third: l = m2 + 1.
Note: Ternary search reduces the interval faster than binary search in terms of splits, but its number of comparisons per step may make it less efficient in practice for simple search (binary search is typically preferred).
