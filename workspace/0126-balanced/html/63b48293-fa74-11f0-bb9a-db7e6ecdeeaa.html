Overfitting Demonstration

This demo shows a simple regression problem with noisy data points generated from a known function. You can fit different polynomial curves to the data by adjusting the polynomial degree using the slider. Underfitting occurs when the model degree is too low and can't capture the true pattern, while overfitting happens when the degree is too high, making the model capture noise as if it were signal.

The plot on the left shows the original noisy data points and the true underlying function (dashed line). The plot on the right shows the fitted polynomial curve for the degree you selected.
