Big-Theta (Θ) Notation — Interactive Demo

Experiment with functions f(n) and g(n) to see whether f(n) ∈ Θ(g(n)) based on sampled values.
About: f(n) ∈ Θ(g(n)) means there exist positive constants c1, c2 and n0 such that for all n ≥ n0: c1·g(n) ≤ f(n) ≤ c2·g(n). This tool samples n values and reports the observed minimum and maximum of the ratio f(n)/g(n). It visualizes f, g and the bounding lines c1·g and c2·g. This is experimental (sampled), not a formal proof.
Quick examples:
3n+5 vs n
2n^2+4n+1 vs n^2
5n log n vs n log n
n vs n^2
log n vs 1
1.5^n vs 2^n
Press Analyze to compute.
—
Notes: No values where g(n) = 0 are accepted. If g becomes nonpositive for sampled n, the tool will warn.
Interpretation: If the ratio f(n)/g(n) stays bounded between two positive constants for all large n, we say f ∈ Θ(g). The plotted horizontal lines on the ratio chart show the observed min and max ratio.
This is a teaching tool. Results are based on discrete sampled points and numeric evaluation.
Implementations use JavaScript's Math functions; use Math.log for natural log.
