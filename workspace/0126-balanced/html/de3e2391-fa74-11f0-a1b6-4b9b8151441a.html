What is a Neural Network?

A neural network is a series of algorithms that attempts to recognize underlying relationships in a set of data through a process that mimics the way the human brain operates. In this visualization:

  • Blue neurons (X1-X3) represent the input layer
  • Green neurons (H1-H2) represent the hidden layer
  • Red neuron (Y) represents the output
  • The connecting lines represent weights between neurons

Each neuron's activation is calculated by taking the weighted sum of inputs from the previous layer, then applying an activation function (here we use sigmoid).
