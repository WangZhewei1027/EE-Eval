JavaScript Indexing Concepts

Array Indexing

Arrays use zero-based indexing in JavaScript (first element is at index 0).

const fruits = ['Apple', 'Banana', 'Cherry', 'Date'];

String Indexing

Strings can also be accessed by index like arrays (read-only).

const message = 'Hello World';

Object Property Indexing

Object properties can be accessed using dot notation or bracket notation (like indexing).

const user = {
    name: 'Alice',
    age: 30,
    email: 'alice@example.com'
};
