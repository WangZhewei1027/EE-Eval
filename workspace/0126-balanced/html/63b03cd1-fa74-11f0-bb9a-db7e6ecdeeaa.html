JavaScript Set Demonstration

What is a Set?

A Set is a built-in JavaScript object that allows you to store unique values of any type. It does NOT allow duplicates.

Basic Operations

// Create a new Set
const mySet = new Set();

// Add some values
mySet.add(1);
mySet.add(5);
mySet.add(5); // duplicate, won't be added
mySet.add('hello');
mySet.add({a:1});

// Display size
console.log('Size:', mySet.size);

// Check for presence
console.log('Has 5?', mySet.has(5));

// Remove a value
mySet.delete(1);

console.log('Values:', Array.from(mySet));

Iterating a Set

// Iterate over values
for (const value of mySet) {
  console.log(value);
}

// Or using forEach
mySet.forEach(value => {
  console.log(value);
});

Use Cases

Examples where Set is useful:

  • Removing duplicate values from an array
  • Checking for existence quickly
  • Tracking unique items (like visited URLs or user IDs)

Remove duplicates from array

// From an array with duplicates
const numbers = [1,2,2,3,4,4,5];

// Create a unique array using Set
const uniqueNumbers = [...new Set(numbers)];

console.log(uniqueNumbers);
