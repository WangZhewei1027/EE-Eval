Instructions: Click on the canvas to add points. Points alternate between two classes: Class 1 (red) and Class 2 (blue). When you have added at least one point of each class, click Train SVM to find the maximum margin hyperplane separating the classes.

The SVM finds the line that best separates the two classes by maximizing the margin, shown by the green parallel lines. Orange highlighted points are the support vectors that define this margin.

Note: This demo implements a simple linear, hard-margin SVM solver using quadratic programming (specifically a small fixed-size QP solver using the quadprog algorithm).
