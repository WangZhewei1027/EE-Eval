Refactoring Demonstration

Refactoring means changing a code's structure without changing its behavior, with the goal of improving readability, maintainability, or performance.

Before Refactoring

This function calculates the total price including tax and a discount, all done in one place:

// Calculate total price with tax and discount function calculateTotalPrice(price, taxRate, discount) { const taxAmount = price * taxRate; const priceWithTax = price + taxAmount; const discountAmount = priceWithTax * discount; const finalPrice = priceWithTax - discountAmount; return finalPrice.toFixed(2); }

After Refactoring

We can improve the code by splitting each step into descriptive functions. This makes the code easier to understand and maintain.

// Calculate tax amount function calculateTax(price, taxRate) { return price * taxRate; } // Calculate discount amount function calculateDiscount(price, discount) { return price * discount; } // Calculate final price with tax and discount function calculateTotalPrice(price, taxRate, discount) { const taxAmount = calculateTax(price, taxRate); const priceWithTax = price + taxAmount; const discountAmount = calculateDiscount(priceWithTax, discount); const finalPrice = priceWithTax - discountAmount; return finalPrice.toFixed(2); }
