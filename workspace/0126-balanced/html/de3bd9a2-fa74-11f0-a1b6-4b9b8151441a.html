What is Exponential Search?

Exponential search is a search algorithm designed for sorted arrays that combines elements of binary search and linear search:

  1. First finds the range where the target element might be by searching for indexes that are powers of 2 (1, 2, 4, 8, etc.)
  2. Once a range is found (where array[i] > target), performs binary search within that range
  3. Works well for unbounded/infinite size arrays
  4. Time complexity is O(log n), making it efficient for large datasets
