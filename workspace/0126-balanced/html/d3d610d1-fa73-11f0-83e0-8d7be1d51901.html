Bubble Sort — Visual Demonstration

Comparisons: 0
Swaps: 0
Bubble Sort iteratively compares adjacent items and swaps them to "bubble" larger values to the end.
Note: You can Step through or Play.
Hover bars to see values. Sorted bars are green. Current comparison is amber. Swapping bars turn red.

Pseudocode

1. for i from 0 to n-1
2.   for j from 0 to n-i-2
3.     if arr[j] > arr[j+1]
4.       swap arr[j], arr[j+1]
5.     end if
6.   end for
7. end for

Controls

- Shuffle: generate a new random array.
- Start: play from current state.
- Pause: pause the animation.
- Step: perform a single comparison/swap step.
- Reset: revert to original unsorted array.
