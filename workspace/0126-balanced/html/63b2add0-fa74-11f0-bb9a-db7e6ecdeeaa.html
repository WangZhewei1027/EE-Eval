CPU Scheduling Algorithms Demonstration

Select a CPU scheduling algorithm and enter processes with their arrival and burst times. The simulation will calculate and visualize scheduling order, waiting times, turnaround times, and a Gantt chart.

Process Input

Enter processes, one per line, in format:
ProcessID ArrivalTime BurstTime [Priority]
Example for Priority Scheduling: P1 0 5 2
Priority is optional and used only in Priority scheduling.
