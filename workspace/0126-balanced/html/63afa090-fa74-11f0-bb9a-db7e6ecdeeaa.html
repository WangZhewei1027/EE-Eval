Dynamic Array Demonstration

This demo simulates a dynamic array (like C++ std::vector) by managing a fixed-size underlying array and resizing (re-allocating) automatically as elements are added beyond its capacity.

How this Dynamic Array works:
- Starts with a small fixed capacity.
- When you add elements, if the current capacity is exceeded, it "resizes" by creating a new internal array (double the capacity), copying existing elements over, and then adding the new element.
- This mimics underlying logic in many dynamic array implementations where capacity grows exponentially to keep amortized insertion efficient.
- You can see the internal capacity and size updates as you add elements.
