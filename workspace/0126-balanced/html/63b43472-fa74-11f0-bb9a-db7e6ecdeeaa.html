Logistic Regression Interactive Demo

Click inside the plot area to add points. There are two classes for points:
Class 1 (Red points) and Class 0 (Blue points).
By default, points you add are Class 0 (Blue). Press the spacebar to toggle the class of the next point you add.
Once you have some points, the model will train continuously and updates the decision boundary in real time.
Logistic Regression tries to find a line that best separates the two classes by estimating probabilities using the logistic sigmoid function.
