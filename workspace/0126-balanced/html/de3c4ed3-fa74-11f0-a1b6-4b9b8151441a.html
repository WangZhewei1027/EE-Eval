What is a Greedy Algorithm?

A greedy algorithm makes the locally optimal choice at each step with the hope of finding a global optimum. It doesn't reconsider previous choices and may not always find the best solution, but it's often efficient.

Coin Change Problem

Given coins of different denominations and a total amount, find the minimum number of coins that make up that amount.

Greedy approach: Always pick the largest coin possible at each step.

Activity Selection Problem

Given a set of activities with start and end times, select the maximum number of activities that can be performed by a single person.

Greedy approach: Always pick the activity with the earliest finish time.
