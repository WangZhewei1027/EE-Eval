Random Forest Concept Demonstration

This visualization demonstrates the concept of Random Forest, an ensemble learning method used for classification. A Random Forest builds multiple decision trees using random subsets of data and features, and combines their results to improve prediction accuracy and control overfitting.

You can generate a random dataset of points belonging to two classes, train multiple decision trees on random bootstrap samples with randomized feature splits, and see how each tree (colored decision regions) votes to classify points in the 2D feature space.
