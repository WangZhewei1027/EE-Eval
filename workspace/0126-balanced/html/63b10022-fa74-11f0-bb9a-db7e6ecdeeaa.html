Merge Sort Demonstration

Merge Sort is a divide-and-conquer sorting algorithm. It divides the array into two halves, recursively sorts each half, and then merges the sorted halves back together.

This visualization shows the process step-by-step:
  • Blue bars represent unsorted elements.
  • Red bars indicate elements currently being compared/merged.
  • Green bars highlight the elements being merged together.
  • Dark green bars mean elements that are finalized in the sorted order.
