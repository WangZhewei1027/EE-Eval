Quick Sort Visualization

How Quick Sort Works:

Quick Sort is a divide-and-conquer algorithm that works as follows:

  1. Choose a pivot element from the array (here we use the last element).
  2. Partition the array so that elements less than the pivot come before it and elements greater come after.
  3. Recursively apply the same process to the sub-arrays before and after the pivot.

In this visualization, red represents the pivot, orange shows elements being compared, and green indicates sorted elements.
