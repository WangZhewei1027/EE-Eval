JavaScript Indexing Demonstration

Indexing in JavaScript is a way to access individual elements of strings or arrays by their position. Positions (indexes) start at 0, meaning the first element has index 0, the second index 1, and so on.

Consider the array:

let fruits = ["Apple", "Banana", "Cherry", "Date", "Elderberry"];

String Indexing

Strings behave like arrays of characters. You can access each character by its index. 

let greeting = "Hello";
console.log(greeting[0]); // "H"
console.log(greeting[4]); // "o"
