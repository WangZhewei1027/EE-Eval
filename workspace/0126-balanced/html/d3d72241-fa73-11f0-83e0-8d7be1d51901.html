Dijkstra's Algorithm — Interactive Demo

Build a weighted graph, choose a source node, and step through Dijkstra's shortest-path process. Visual cues show which nodes are in the fringe, currently processed, or finalized.

Mode: Drag / Select. Click nodes to inspect. Use "Set Source" to pick the start node.

How it works: Dijkstra repeatedly selects the closest non-final node (the fringe), relaxes its outgoing edges (updating distance estimates), and marks it finalized when done. Use Next to step one atomic action (extract or relax one neighbor).
Click canvas to add nodes (when in Add Node). For adding an edge: click a node then another (when in Add Edge). Drag nodes when in Drag mode.

Distances, previous pointers, and the fringe (nodes reachable but not finalized) are shown. The priority queue is represented by the fringe ordered by current distance.
