Binary search repeatedly divides a sorted array in half. Compare the target with the middle element: if equal — found; if smaller — continue on the left half; if larger — continue on the right half. It runs in O(log n) time for n elements. 
function binarySearch(a[], target):
  low = 0
  high = length(a)-1
  while low <= high:
    mid = floor((low + high) / 2)
    if a[mid] == target:
      return mid
    else if a[mid] < target:
      low = mid + 1
    else:
      high = mid - 1
  return -1
Time Complexity: Best O(1) if found at mid, Average and Worst O(log n). Space: O(1) iterative, O(log n) recursion stack.
