Priority Queue (frontier)

Pseudocode (highlights current step)

1: Select any start vertex s, mark visited(s) = true
2: Insert all edges (s, v) into priority queue by weight
3: While PQ not empty and not all vertices visited:
3.1: Extract edge (u, v) with min weight
3.2: If visited(v) continue (skip)
3.3: Add (u,v) to MST, mark visited(v) = true
3.4: Insert all edges (v, x) where visited(x)=false into PQ

Notes

  • Click a node to choose the starting vertex before "Init".
  • Press "Init Prim" to begin. Then use "Step" to advance one edge selection at a time.
  • "Auto" runs steps every 700ms. "Reset" clears algorithm (keeps graph).
  • The demo ensures the graph is connected on generation.
