Interpolation Search — Interactive Demo

Interpolation search estimates the likely position of the target using a linear interpolation formula. It can be faster than binary search on uniformly distributed, sorted data.

low high pos (probe) found
pos = low + ((target - A[low]) * (high - low)) / (A[high] - A[low])
Steps: Use Step to manually advance or Start to run automatically. The demo shows each probe (pos) and updates low/high accordingly. If A[high] == A[low], interpolation falls back to direct check.
