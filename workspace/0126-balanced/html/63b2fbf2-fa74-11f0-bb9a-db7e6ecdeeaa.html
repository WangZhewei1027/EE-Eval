What is a B-Tree Index?

A B-Tree index is a balanced tree data structure that maintains sorted data and allows searches, sequential access, insertions, and deletions in logarithmic time.

It is widely used in databases and file systems to efficiently store and retrieve data. In a B-Tree:

  • Each node can contain multiple keys (between a minimum and maximum degree).
  • Nodes have children pointers defining subtrees between keys.
  • The tree remains balanced by splitting and merging nodes during insertions and deletions.

This visualization demonstrates inserting keys, how the B-Tree nodes split when they overflow, keeping the tree balanced and indexed.
