Big-Theta Notation

Big-Theta notation is used to describe the upper bound of an algorithm's time complexity.

It is a way to express the relationship between the number of operations performed by an algorithm and the size of the input.

It is usually denoted by the symbol O and is often used to describe the time complexity of algorithms.

Examples of Big-Theta notation include:

  • O(1)
  • O(log n)
  • O(n)
  • O(n log n)
