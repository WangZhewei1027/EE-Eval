What is Big-Omega?

Big-Omega (Ω) notation describes a lower bound on the growth rate of a function. It tells us the minimum rate at which a function grows as its input size becomes large. Formally, f(n) ∈ Ω(g(n)) means constants c > 0 and n₀ exist such that for all n ≥ n₀,

f(n) ≥ c × g(n)

In simpler terms, beyond some input size, f(n) is always greater than or equal to c times g(n).

  • g(n) = n (linear function)
