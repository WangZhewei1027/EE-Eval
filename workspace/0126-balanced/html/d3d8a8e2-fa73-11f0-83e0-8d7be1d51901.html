What's happening?
Select a demo and run operations. Watch the cost each operation (bars) and the running average (green line). In amortized analysis we ask: over a sequence of operations, what is the average cost per operation in the worst case?
Amortized techniques: aggregate, accounting, potential. Examples here visualize aggregate (total cost / #ops) and show why doubling gives amortized O(1).
Chart: blue bars = cost of each operation (vertical). Green line = running average (amortized cost).
