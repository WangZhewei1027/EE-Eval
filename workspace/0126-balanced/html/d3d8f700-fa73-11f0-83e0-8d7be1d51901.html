Summary

How it works

This visualizer simulates CPU scheduling algorithms. It produces a Gantt chart with contiguous CPU segments. Metrics: - Turnaround = Completion - Arrival - Waiting = Turnaround - Burst - Response = First Start - Arrival For preemptive algorithms a process can have multiple CPU segments; the first start is used for response time. Idle intervals are shown when CPU is waiting for the next arrival.
Created as a single-file demonstration. Modify processes, select algorithm, and explore scheduling behavior.
