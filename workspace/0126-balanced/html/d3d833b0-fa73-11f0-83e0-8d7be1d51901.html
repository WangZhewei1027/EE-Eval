// Sliding window (fixed-size)
let windowSum = sum(arr[0..k-1]);
let maxSum = windowSum;
for (let i = k; i < arr.length; i++) {
  windowSum += arr[i] - arr[i-k]; // slide right by 1
  if (windowSum > maxSum) update max;
}

The sliding window technique maintains a subset (window) of contiguous elements and moves it (slides) over the input to compute values incrementally: - Fixed-size: maintain sum of exactly k elements — update by adding entering element and subtracting leaving element. - Variable-size: expand right pointer, shrink left pointer when constraint violated (e.g., duplicates or sum constraint).
- Max sum of k: [2,1,5,1,3,2], k=3 -> best sum 9 for window [5,1,3].
- Longest unique substring: "abcabcbb" -> "abc" length 3.
