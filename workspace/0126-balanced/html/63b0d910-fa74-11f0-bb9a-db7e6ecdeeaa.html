About Suffix Trees

A Suffix Tree is a compressed trie containing all the suffixes of a given text as their keys. It allows many useful string operations to be done quickly, such as substring search, pattern matching, and repeated substring detection.

The tree displayed shows nodes (blue circles for internal nodes, green for leaves). Edges are labeled with substrings of the input. Each leaf represents a suffix position — the number in a leaf node is the start index of that suffix.

The input string is automatically appended with a unique terminal character $ to ensure all suffixes end uniquely.

For more info, see Wikipedia - Suffix Tree.
