Big-Omega (Ω) Notation

Definition

Big-Omega (Ω) notation describes the lower bound of an algorithm's growth rate. It represents the minimum resources (time or space) an algorithm needs as the input size grows.

Mathematically, we say f(n) = Ω(g(n)) if there exist positive constants c and n₀ such that:

f(n) ≥ c·g(n) for all n ≥ n₀

Common Examples

  • Linear search: Ω(1) - best case when item is first
  • Binary search: Ω(1) - best case when item is middle
  • Bubble sort: Ω(n) - when list is already sorted
  • Matrix multiplication: Ω(n²) - must touch all elements
