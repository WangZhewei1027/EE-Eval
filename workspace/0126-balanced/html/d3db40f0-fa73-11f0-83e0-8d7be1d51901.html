The runtime provides built-in APIs. Some exist in browsers, others only in Node.js.
  • Browser runtime: DOM (document), fetch, localStorage, Web Workers, WebSockets, Web Crypto.
  • Node.js runtime: filesystem (fs), process, Buffer, native modules, global require.
  • Common: memory (heap), call stack, scheduler/event loop, garbage collector, globalThis.
