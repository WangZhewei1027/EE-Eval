Tiny Compiler — interactive demo

This page demonstrates the main compiler stages on a tiny language: lexical analysis (tokenize), parsing (AST), a simple optimizer (constant folding + propagation), code generation to JavaScript, and execution. Type or paste a TinyLang program on the left and click the buttons to run each stage. The language supports: variable declarations (let), arithmetic, and print(). Example: let x = 2 + 3 * (4 - 1); print(x);
