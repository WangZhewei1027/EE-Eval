JavaScript Dynamic Typing Demonstration

JavaScript is a dynamically typed language, which means a variable can hold values of any type, and that type can change at runtime. 

// Declare a variable with a number
let dynamicVar = 42;
console.log(typeof dynamicVar, dynamicVar); // number 42

// Assign a string to the same variable
dynamicVar = "Now I'm a string!";
console.log(typeof dynamicVar, dynamicVar); // string "Now I'm a string!"

// Assign a boolean to the same variable
dynamicVar = true;
console.log(typeof dynamicVar, dynamicVar); // boolean true

// Assign an object to the same variable
dynamicVar = { message: "I'm an object now" };
console.log(typeof dynamicVar, dynamicVar); // object { message: "I'm an object now" }
