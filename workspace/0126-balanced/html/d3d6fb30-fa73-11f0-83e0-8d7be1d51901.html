Algorithm explanation
Exponential Search first finds a range where the target could appear by checking indices 1, 2, 4, 8, ... (doubling each time) until a value greater than or equal to the target is found or the end of array is reached. Then it runs a binary search in the identified range. It is especially effective for unbounded or exponential-growing data.
Time complexity: O(log i) where i is the position of the target (plus binary search within the found range). Worst-case O(log n).
