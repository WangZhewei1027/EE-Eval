Bellman-Ford Algorithm — Interactive Demo
Build or use a sample directed graph (weights can be negative). Step through the Bellman-Ford algorithm to compute shortest paths from a chosen source. The algorithm relaxes all edges V-1 times and can detect negative-weight cycles.
Instructions:
  • Click canvas to add nodes (or use Add Node).
  • Click a node then another to create a directed edge (you'll be prompted for weight).
  • Drag nodes to reposition. Use Initialize to pick source and start.
  • Use Step to relax the next edge, Next Iteration to finish current pass, Auto to play.
