Refactoring Demonstration

Refactoring is the process of restructuring existing code without changing its external behavior. Below is an example of refactoring a simple function.

Original Code

function calculateTotal(items) { let total = 0; for (let i = 0; i < items.length; i++) { if (items[i].price > 100) { total += items[i].price * 0.9; // 10% discount for expensive items } else { total += items[i].price; } } return total; }

Refactored Code

function calculateTotal(items) { return items.reduce((total, item) => { return total + (item.price > 100 ? item.price * 0.9 : item.price); }, 0); }

Refactoring Benefits:

  • More concise and expressive code
  • Uses modern JavaScript features (reduce)
  • Eliminates temporary variables
  • Easier to read and maintain
  • Same functionality with less code
