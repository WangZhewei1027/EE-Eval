for i = 1 to n-1
    key = A[i]
    j = i - 1
    while j >= 0 and A[j] > key
        A[j+1] = A[j]   // shift right
        j = j - 1
    A[j+1] = key       // insert key
Complexity: Best O(n) (already sorted), Average/Worst O(n^2). Stable: Yes. In-place: Yes.
