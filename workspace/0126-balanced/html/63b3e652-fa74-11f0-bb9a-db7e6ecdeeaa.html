About the Interpreter Pattern

    

      The Interpreter Pattern is a behavioral design pattern that defines a grammatical representation for a language,
      and provides an interpreter to deal with this grammar.
    

    

      In this demo, we interpret simple arithmetic expressions containing addition and subtraction of integers.
      Each number or operation is represented by an Expression object.
      The interpret() method executes the expression and returns a result.
    

    

      The key concept is to represent sentences or expressions as a tree of expressions, each knowing how to interpret itself.
