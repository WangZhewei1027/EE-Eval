Hash Functions — Interactive Demo

This single-page demo shows several simple hash functions, the cryptographic SHA-256, collision behavior, and how hashing maps keys into buckets (hash table). Use the controls to explore properties like one-wayness and the avalanche effect.

4. Hash Table Visualization

How hash functions map keys into a fixed set of buckets (indices). Insert keys and watch how collisions are stored (chaining).
Short notes: - Toy hashes here (sum, djb2, polynomial) are used for learning — they are not cryptographically secure. - SHA-256 is a cryptographic hash: long fixed-size output (256 bits), strong avalanche and one-way properties.
