SQL Concept Demonstration

What is SQL?

SQL (Structured Query Language) is a standard language for managing and manipulating databases. It allows you to create, read, update, and delete data in a relational database.

Sample SQL Table: Employees

Imagine a table with employee data:

CREATE TABLE Employees (
  ID INT PRIMARY KEY,
  Name VARCHAR(100),
  Department VARCHAR(50),
  Salary INT
);

And some sample data:

IDNameDepartmentSalary
1AliceEngineering75000
2BobMarketing50000
3CharlieEngineering80000
4DanaHR60000

Common SQL Queries

Here are a few example SQL queries & their explanations:

1. Select all employees

SELECT * FROM Employees;

This fetches all columns and all rows from the Employees table.

2. Select employees in Engineering

SELECT Name, Salary FROM Employees WHERE Department = 'Engineering';

This fetches only the Name and Salary of employees working in the Engineering department.

3. Increase salary by 10% for Marketing department

UPDATE Employees SET Salary = Salary * 1.1 WHERE Department = 'Marketing';

This updates the salary for Marketing employees by increasing it by 10%.

4. Delete employee with ID = 4

DELETE FROM Employees WHERE ID = 4;

This deletes the employee Dana from the table.

5. Aggregate example: Average salary by department

SELECT Department, AVG(Salary) AS AvgSalary
FROM Employees
GROUP BY Department;

This calculates the average salary per department.
