Deque (Double-ended Queue) Demonstration

A deque (double-ended queue) is a data structure that allows insertion and removal of elements from both ends. It's more flexible than both stacks (LIFO) and queues (FIFO).

Operations:

  • addFront(item): Add to front
  • addRear(item): Add to back
  • removeFront(): Remove from front
  • removeRear(): Remove from back
  • peekFront(): See front item
  • peekRear(): See back item
  • size(): Get current size
  • isEmpty(): Check if empty
