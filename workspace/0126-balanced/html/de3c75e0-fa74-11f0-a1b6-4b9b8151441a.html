Sliding Window Algorithm Demo

Maximum Sum of Subarray of Size K

Find the maximum sum of any contiguous subarray of size 3.

Current maximum sum: 0

How it works:

  1. Initialize window sum and max sum with the sum of first window
  2. Slide the window by one element at a time
  3. Subtract the element going out of the window
  4. Add the new element entering the window
  5. Update max sum if current window sum is greater
