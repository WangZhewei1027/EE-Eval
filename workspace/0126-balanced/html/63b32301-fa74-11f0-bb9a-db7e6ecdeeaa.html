Transaction Concept Demonstration

This demo simulates a bank transfer transaction between two accounts. A transaction groups multiple operations in an atomic way:

  • Either all operations succeed (commit), or
  • None are applied (rollback) if any operation fails
Here, we simulate this by trying to debit Account A and credit Account B. If debit fails (e.g. insufficient funds), transaction aborts and no balance changes apply.
