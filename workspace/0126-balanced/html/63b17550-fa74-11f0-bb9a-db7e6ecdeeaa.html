Ternary Search Visualization

Ternary search is a divide and conquer algorithm that works on sorted arrays. It splits the search interval into three parts using two mid indices and determines which part the target element lies in, then repeats the process on that segment.

It is similar to binary search but divides the array into three parts instead of two.

Algorithm steps:

  1. Calculate two mid points: mid1 and mid2.
  2. If target equals array[mid1] or array[mid2], return the index.
  3. If target is less than array[mid1], search in the left segment.
  4. If target is greater than array[mid2], search in the right segment.
  5. Otherwise search in the middle segment.
  6. Repeat until element is found or interval is empty.
