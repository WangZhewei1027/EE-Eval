Correlation Analysis: Three Evaluation Frameworks vs Human Judgment

Workspace: 0126-biased | Sample Size: N=77

Executive Summary

Objective: Compare three automated evaluation frameworks against human judgment across four quality dimensions.

Sample Size: 77 HTML pages with complete evaluation data from all frameworks.

Statistical Method: Pearson correlation coefficient with significance testing (*** p<0.001, ** p<0.01, * p<0.05).

Evaluation Frameworks

FSM-based Framework: Structural behavior analysis using finite state machines. Score = average of three dimensions (structural similarity, semantic similarity, isomorphism similarity). Range: 0-1.
VLM Framework (Baseline 1): Vision-Language Model evaluation using GPT-4 Vision API. Score = visual_quality rating. Range: 0-5.
Playwright Framework (Baseline 2): DOM-driven functional testing. Score = model-level test pass rate (mean across all tests for each model). Range: 0-1.
Human Evaluation: Manual ratings by 2 annotators across 4 dimensions (Functional, Visual, Interactivity, Pedagogical). Range: 0-5.
